# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Erste ein Berechtigungsfehler beim Öffnen eines freigegebenen Postfachs in Outlook im Web?

Wenn Sie die Fehlermeldung **mögliche etwas Fehlerursache. Sie sind nicht berechtigt, dieses Postfach zu öffnen.** Wenn Sie versuchen, ein freigegebenes Postfach in Outlook im Web zu öffnen, möglicherweise Ihr Konto nicht über die Berechtigung zum Zugriff auf das freigegebene Postfach. Administrator Ihrer Organisation kann Ihren Kontoberechtigungen gemäß die Anweisungen im Thema [Erstellen eines freigegebenen Postfachs](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9)für das freigegebene Postfach zuweisen.
