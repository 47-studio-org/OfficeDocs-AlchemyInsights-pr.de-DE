# <a name="a-user-has-two-mailboxes"></a>Ein Benutzer hat zwei Postfächer

Hybridumgebungen, mit dem Azure Active Directory verbinden (AAD verbinden) oder DirSync möglicherweise versehentlich einen Benutzer zwei Postfächer haben: eine in der lokalen Organisation und eine in der Cloud.
  
Zum Beheben dieses Problems finden Sie unter [Dauerhaft löschen vorherigen Postfach Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
