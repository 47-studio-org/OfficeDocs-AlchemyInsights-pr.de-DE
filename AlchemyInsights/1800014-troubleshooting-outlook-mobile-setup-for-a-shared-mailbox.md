# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Problembehandlung bei Outlook mobile Setup für freigegebene Postfächer

Sie können nicht das freigegebene Postfach aus der Outlook-app zugreifen, die auf dem mobilen Gerät installiert ist. Möchten Sie dieses Feature? Lassen Sie die Outlook-Entwicklungsteams kennen. [Stimme hier](https://go.microsoft.com/fwlink/?linked=862116)!
  
Sie können nur einem freigegebenen Postfach auf einem mobilen Gerät die Open Browser Anmelden bei Office 365, und wechseln Sie Outlook im Web. Anweisungen finden Sie unter [Hinzufügen eines freigegebenen Postfachs in Outlook im Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Aus Outlook im Web werden Sie das freigegebene Postfach zugreifen können.
