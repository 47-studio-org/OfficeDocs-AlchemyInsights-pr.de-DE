# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Verwenden Sie SaRA zum Diagnostizieren und Beheben von Problemen bei der Authentifizierung mit Outlook

Es wird dringend empfohlen, mit denen Sie die [Outlook behält die Zuordnung für mein Kennwort gefragt](https://aka.ms/SaRA-OutlookPwdPrompt) diagnostic auf dem betreffenden Computer zum Behandeln von Problemen, auf dem Outlook ständig zur Eingabe eines Kennworts aufgefordert. Diese [SaRA](https://diagnostics.office.com/#/) diagnostic automatisierte überprüft werden und gibt Lösungsvorschläge gefundene Probleme beheben können.
