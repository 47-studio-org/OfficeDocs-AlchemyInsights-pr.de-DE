# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>Frei/Gebucht-Informationen Kalenderproblemen in hybridumgebungen

Wenn Benutzer des Kalenders Frei/Gebucht-Informationen in hybridumgebungen Probleme auftreten, führen Sie den Assistenten für Hybridkonfigurationen erneut aus. Weitere Informationen zum Assistenten für die Hybridkonfiguration finden Sie unter [Hybrid-Konfigurations-Assistenten](https://go.microsoft.com/fwlink/p/?linkid=528149).
  
Schritte zur Problembehandlung in hybridumgebungen, finden Sie unter [Problembehandlung bei einer hybridbereitstellung](https://technet.microsoft.com/library/jj659053.aspx).
