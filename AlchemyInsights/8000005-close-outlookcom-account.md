# <a name="close-your-outlookcom-email-account"></a>Schließen Sie Ihr Outlook.com e-Mail-Konto

1. Wechseln Sie zur Seite [Ihr Konto zu schließen](https://go.microsoft.com/fwlink/p/?linkid=845493) . 
    
2. Befolgen Sie die Anweisungen auf der Seite, um sicherzustellen, dass Ihr Konto geschlossen wird bereit ist, und wählen Sie **Weiter**aus. 
    
3. Lesen Sie die Liste, und aktivieren Sie die Kontrollkästchen, um zu bestätigen, dass Sie jedes Element gelesen haben.
    
4. Wählen Sie den Grund, dass Sie das Konto schließen möchten, in der Dropdownliste **Wählen Sie einen Grund** . 
    
5. Wählen Sie **Mark Konto für schließen**. 
    
Weitere Informationen finden Sie unter [Outlook.com Konto stilllegen](https://go.microsoft.com/fwlink/p/?linkid=873106)[](https://support.office.com/article/564b801e-2a47-4cb2-afa8-12ead3185038.aspx).
