# <a name="change-your-profile-name"></a>Ändern der Profilname

1. Fahren Sie mit [Ihren Microsoft-Kontoinformationen](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Wählen Sie **Name bearbeiten**. 
    
3. Geben Sie einen Vorname und Nachname, und wählen Sie **Speichern**aus. 
    
Warten Sie bis zu 24 Stunden für Änderungen an Ihren Profilnamen wirksam wird.
  
Weitere Informationen finden Sie finden Sie unter [Ändern Ihrer Profilfotos oder den Namen in Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
