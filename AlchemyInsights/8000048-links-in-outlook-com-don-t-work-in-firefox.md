# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Links in Outlook.com funktionieren nicht in Firefox

Links in e-Mail-Nachrichten Outlook.com funktionieren nicht, wenn sie in Firefox-Browser angezeigt werden.
  
Mozilla ist für Firefox, jedoch erst dann [an eine Behebung arbeiten](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) : 
  
- Verwenden Sie [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) oder einen anderen Webbrowser. 
    
- Verwenden Sie die Outlook.com Beta. [Erfahren Sie mehr über die Betaversion Outlook.com.](https://go.microsoft.com/fwlink/p/?linkid=874356&amp;clcid=0x409)
    
- Anzeigen von e-Mail im Lesebereich. Weitere Informationen finden Sie unter Gewusst wie: [Ändern der Darstellung von Ihr Postfach Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001401&amp;clcid=0x409).
