# <a name="create-a-new-folder"></a>Erstellen Sie einen neuen Ordner

1. Wählen Sie unter **Ordner** im Navigationsbereich **neuen Ordner**. 
    
2. Geben Sie einen Namen für den Ordner ein, und drücken Sie die EINGABETASTE.
    
Finden Sie weitere Informationen finden Sie unter [Erstellen eines Ordners in Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
