# <a name="how-to-change-your-theme"></a>Gewusst wie: Ändern Ihres Designs

Um das Design in Outlook.com oder in Outlook im Web zu ändern, rufen Sie die **Einstellungen**![Einstellungen](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) , und wählen Sie ein Design unter **Design**. Um alle verfügbaren Designs angezeigt wird, wählen Sie **Alle Ansicht** unter **Design**aus.
