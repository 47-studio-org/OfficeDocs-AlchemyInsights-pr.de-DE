# <a name="turn-off-focused-inbox"></a>Posteingang praxisorientierte deaktivieren

Fokussierte Posteingang hilft bei die Sie verpassen konzentrieren. E-Mail, die Sie möglicherweise zum Lesen von wird unter spezifische Darstellung angezeigt, und die restlichen auf andere verschoben wird. Zu erfahren, wie es funktioniert, [Lesen Sie diese](https://support.office.com/article/f445ad7f-02f4-4294-a82e-71d8964e3978#bkmk_consumer). Wenn Sie nicht praxisorientierte Posteingang verwenden möchten, wählen Sie unten auf die Schaltfläche.
