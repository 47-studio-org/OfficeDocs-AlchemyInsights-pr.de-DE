# <a name="how-to-right-click-on-a-mac"></a>Wie Sie auf einem Mac mit der rechten Maustaste

Auf Mac-Computern wird mit der rechten Maustaste auf sekundären klicken oder CTRL genannt. Wenn die Maus, Trackpad oder andere Eingabegeräte eine Schaltfläche mit der rechten Maustaste oder eine andere Möglichkeit zum Ausführen eines Rechtsklick enthält, halten Sie die **STRG-Taste** auf der Tastatur gedrückt, während Sie klicken. 
  
Weitere Informationen finden Sie unter [Apple unterstützen](https://go.microsoft.com/fwlink/?linkid=2022220&amp;clcid=0x409).
