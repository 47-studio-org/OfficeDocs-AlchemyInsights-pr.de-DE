# <a name="internal-server-errors-in-outlookcom"></a>Interne Serverfehler in Outlook.com

Oh nein! Wenn die Meldung "Fehler 500" oder "interner Server Fehler" angezeigt wird, sind Sie es uns und nicht Ihnen leid.

Sehen Sie sich die [Seite Office-Dienst Status](https://portal.office.com/servicestatus) an, um sicherzustellen, dass es nichts anderes gibt, aber die beste Lösung besteht darin, eine Pause für ein paar Minuten zu machen und dann Ihren Browser zu aktualisieren, indem Sie **STRG** und **F5**drücken.
