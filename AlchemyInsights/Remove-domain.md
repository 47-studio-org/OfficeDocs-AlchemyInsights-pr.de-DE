# <a name="trying-to-remove-your-domain"></a>Möchten Sie Ihre Domäne entfernen?

Führen Sie die Schritte unter [Entfernen einer Domäne](https://docs.microsoft.com/office365/admin/get-help-with-domains/remove-a-domain) aus.
  
Sie müssen alle Benutzer oder Gruppen, die diesem Domänennamen zugeordnet sind, aktualisieren, bevor Sie die Domäne entfernen. In den [häufig gestellten Fragen zu Domänen](https://docs.microsoft.com/office365/admin/setup/domains-faq) finden Sie weitere häufig verwendete Lösungen.
