# <a name="access-denied-due-to-ip-restriction"></a>Zugriff verweigert aufgrund von IP-Einschränkungen

Als IT-Administrator können Sie den Zugriff auf SharePoint-und OneDrive-Ressourcen in Microsoft 365 basierend auf definierten Netzwerkstandorten, denen Sie Vertrauen, Steuern. Dies wird auch als standortbasierte Richtlinie bezeichnet. Weitere Informationen finden Sie unter [Steuern des Zugriffs basierend auf dem Netzwerkspeicherort](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location).
