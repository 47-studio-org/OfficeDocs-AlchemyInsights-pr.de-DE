# <a name="fixing-the-office-apps-sorry-another-account-from-your-organization-is-already-signed-in-message"></a>Beheben der Office-Apps "Sorry, ein anderes Konto aus Ihrer Organisation ist bereits angemeldet" Nachricht

Versuchen Sie Folgendes, um diesen Fehler zu beheben:

1. Öffnen Sie eine Office-App, und [melden Sie sich](https://support.office.com/article/5a20dc11-47e9-4b6f-945d-478cb6d92071) bei vorhandenen Benutzerkonten ab.   
2. Entfernen Sie mithilfe von Windows- **Einstellungs** > **Konten** > **e-Mail-#a0 Konten**alle Arbeits Konten außer dem betroffenen Konto. 
3. Verwenden von Windows- **Einstellungen** > **-Konten** > **Zugriff auf Arbeit oder Schule**: Trennen Sie alle Arbeits Konten mit Ausnahme des betroffenen Kontos. 
4. Office-Aktivierungsstatus zurücksetzen. [Anleitung](https://docs.microsoft.com/office365/troubleshoot/activation/reset-office-365-proplus-activation-state
).
5. [Melden Sie sich](https://support.office.com/article/628ea040-f265-49de-b986-be09c3ebf8a9) mit dem betroffenen Benutzerkonto an. 

Weitere Informationen finden Sie unter ["Sorry, ein anderes Konto aus Ihrer Organisation ist bereits auf diesem Computer angemeldet" in Office 2013](https://docs.microsoft.com/office/troubleshoot/error-messages/another-account-already-signed-in).

Für Mac finden Sie unter [CAN 't Sign in an an a Office 2016 for Mac app](https://docs.microsoft.com/office365/troubleshoot/authentication/sign-in-to-office-2016-for-mac-fail).
