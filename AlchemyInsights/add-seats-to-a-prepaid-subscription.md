# <a name="add-seats-to-a-prepaid-subscription"></a>Hinzufügen von Arbeitsplätzen zu einem vorausbezahlten Abonnement

1. Wechseln Sie im Admin Center zur Seite [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), oder wählen Sie **Abrechnung** \> **Abonnements** aus.
    
2. Wählen Sie das vorausbezahlte Abonnement, das Sie Benutzer hinzufügen möchten.
    
3. Wählen Sie im Menü **Weitere Aktionen** **Erneuern oder Lizenz mit Token hinzufügen**.
    
4. Wählen Sie die Option **neue und nicht verwendeten Product Keys** . (Wenn dieses Abonnement mit einem Product Key im Einzelhandel erworben erstellt wurde, kann die Option zum Hinzufügen von Benutzern mit anderen Zahlungsmethoden müssen.) 
    
5. Wählen Sie **Weiter** aus.
    
6. Geben Sie den Product Key ein, und wählen Sie dann **Weiter**.
    
7. Wählen Sie die Option weitere Benutzer hinzufügen, wählen **Weiter**, und wählen Sie dann **einlösen**.
