# <a name="deploy-microsoft-365-apps"></a>Bereitstellen von Microsoft 365-apps

Für die Bereitstellung von Microsoft 365-Apps für mehrere Endbenutzer über das Office-Bereitstellungs Tool, den Microsoft Endpoint Configuration Manager, als Teil eines Betriebssystem-Festplattenabbilds oder in einem Szenario mit einem gemeinsam genutzten Computer mit Windows Server mit Remote Desktop Dienste (RDS) oder Terminal Diensten sollten Sie die Schritt-für-Schritt-Anleitung im **Microsoft 365 apps for Enterprise-Bereitstellung**
  
Stellen Sie zunächst sicher, dass Sie als mandantenadministrator oder globaler Administrator angemeldet sind, und klicken Sie dann auf hier: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
