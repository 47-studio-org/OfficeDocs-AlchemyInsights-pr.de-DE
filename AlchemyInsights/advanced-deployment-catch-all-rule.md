# <a name="deploy-office-apps"></a>Bereitstellen von Office-Apps

Wenn Sie Office-Apps für mehrere Endbenutzer über das Office-Bereitstellungs Tool, den Microsoft Endpoint Configuration Manager, als Teil eines Betriebssystem-Festplattenabbilds oder in einem Szenario mit Windows Server mit Remote Desktop Dienste (RDS) oder Terminal Diensten bereitstellen möchten, sollten Sie die Schritt-für-Schritt-Anleitung im **Microsoft 365 apps for Enterprise-Bereitstellungs Ratgeber**verwenden.
  
Stellen Sie zunächst sicher, dass Sie als mandantenadministrator oder globaler Administrator angemeldet sind, und klicken Sie dann auf hier: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
