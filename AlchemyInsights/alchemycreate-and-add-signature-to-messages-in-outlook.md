# <a name="creating-email-signatures"></a>Erstellen von e-Mail-Signaturen

Diese Artikel sollen Ihnen helfen, die e-Mail-Signaturen zu erstellen, die Sie suchen.
  
- Ich möchte eine e-Mail-Signatur mithilfe von Outlook unter Windows erstellen: [Erstellen und Hinzufügen einer Signatur zu Nachrichten](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)
    
- Ich möchte eine e-Mail-Signatur mithilfe von Outlook im Web erstellen: [Erstellen und Hinzufügen einer e-Mail-Signatur in Outlook im Web](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)
    
- Ich möchte eine e-Mail-Signatur mit Outlook.com erstellen: [Erstellen und Hinzufügen einer e-Mail-Signatur in Outlook.com](https://support.office.com/article/776d9006-abdf-444e-b5b7-a61821dff034.aspx)
    
- Ich möchte eine globale e-Mail-Signatur für alle e-Mails erstellen, die von meinem Unternehmen gesendet werden: [Erstellen von organisationsweiten Signaturen und Haftungsausschlüssen in Office 365](https://support.office.com/article/2d75860f-c527-4352-a7f6-73eba54c0c72.aspx)
    
 **Tipp:** Organisationsweite Signaturen werden als "Haftungsausschlüsse" bezeichnet.
