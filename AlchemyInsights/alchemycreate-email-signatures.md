# <a name="create-email-signatures"></a>Erstellen von e-Mail-Signaturen

In den folgenden Themen finden Sie Informationen zum Erstellen von e-Mail-Signaturen in Outlook.
  
- Erstellen Sie eine e-Mail-Signatur für Ihre eigenen Nachrichten: [Erstellen und Hinzufügen einer Signatur zu Nachrichten](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) oder [Erstellen und Hinzufügen einer e-Mail-Signatur in Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx).
    
- Erstellen Sie eine globale e-Mail-Signatur für alle von Ihrem Unternehmen gesendeten Nachrichten (auch als Haftungsausschluss bezeichnet): [organisationsweite Nachrichten Ausschlüsse, Signaturen, Fußzeilen oder Kopfzeilen](https://go.microsoft.com/fwlink/p/?linkid=391096).
