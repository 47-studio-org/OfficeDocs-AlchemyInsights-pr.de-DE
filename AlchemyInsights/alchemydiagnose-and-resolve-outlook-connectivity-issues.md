# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnostizieren und Beheben von Outlook-Verbindungsproblemen

Zum Diagnostizieren und Beheben von Outlook-Verbindungsproblemen können Sie das [Outlook-Verbindungstool](https://aka.ms/SaRA-OutlookDisconnect) auf dem betroffenen Computer ausführen. 
  
Dieses Tool führt Diagnoseprüfungen durch und schlägt mögliche Lösungen vor, um das Outlook-Verbindungsproblem zu beheben.
