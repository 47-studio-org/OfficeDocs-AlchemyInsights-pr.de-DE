# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708-Zugriff verweigert, Datenverkehr wird von dieser IP-Adresse nicht akzeptiert

Dieser Fehler tritt auf, wenn Sie e-Mails von bekannten, schwachen Ruf-IP-Adressen senden, die in der Regel von neuen Kunden verwendet werden.
  
Wenn Sie e-Mails von diesen IP-Adressen senden müssen, bevor Sie Lizenzen erwerben können, wenden Sie sich an den Support, um eine Ausnahme anzufordern, bis Sie Lizenzen erwerben können.
