# <a name="assign-unique-permissions-in-sharepoint"></a>Zuweisen eigener Berechtigungen in SharePoint

Wenn Sie die Vererbung von Berechtigungen für eine Liste, eine Bibliothek oder eine Umfrage unterbrechen, und dann neue Berechtigungseinstellungen definieren, wird die Liste (oder Bibliothek) das übergeordnete Element für die Elemente, die in ihr enthalten sind.  

Zum Unterbrechen der Vererbung und Zuweisen eigener Berechtigungen gehen Sie wie folgt vor: [Zuweisen eigener Berechtigungen in SharePoint](https://support.microsoft.com/office/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782#bkmk_break).
