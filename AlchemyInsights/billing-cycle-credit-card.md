# <a name="billing-cycle-credit-card"></a>Abrechnung Cycle Kreditkarte

Wenn Sie direkt über Microsoft Office 365 für Unternehmen Abonnement kaufen, können Sie mit einer Kreditkarte Zahlen. In einigen Ländern oder Regionen können Sie auch mit einem Bankkonto bezahlen.
  
Wenn Sie entscheiden, eine Kreditkarte als die Zahlungsmethode verwenden, können Sie erfahren Sie, wie Verwaltung und Problembehandlung von [Hinzufügen, aktualisieren oder Entfernen einer Kreditkarte oder Bankkonto in Office 365 für Unternehmen](https://support.office.com/article/30ba9c83-50d8-4020-90ed-830a5b8c8724?wt.mc_id=billing_cycle_AI).
  
Dazu gehören Beheben von Problemen, wenn Ihre Kreditkarte abgelehnt wird und Kreditkarteninformationen geändert, sodass die Karte akzeptiert wird.
