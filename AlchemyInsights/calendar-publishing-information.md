# <a name="calendar-publishing-information"></a>Kalender Veröffentlichungsinformationen

- Microsoft 365-Administratoren können die Kalenderveröffentlichung und die Kalenderfreigabe für eine Organisation aktivieren oder deaktivieren. Weitere Informationen finden Sie unter [Aktivieren der Kalenderveröffentlichung, damit Personen Informationen freigeben](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC) und [Kalender für externe Benutzer freigeben](https://docs.microsoft.com/microsoft-365/admin/manage/share-calendars-with-external-users)können.

- Outlook-Benutzer können Ihre eigenen Kalender freigeben. Weitere Informationen finden Sie unter [Freigeben eines Outlook-Kalenders für andere Personen](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88).
