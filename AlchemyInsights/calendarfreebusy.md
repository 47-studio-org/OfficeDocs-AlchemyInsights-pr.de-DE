# <a name="issues-seeing-calendar-freebusy-information"></a>Probleme beim Anzeigen von Kalender-Frei/Gebucht-Informationen

Wenn Sie keine Kalender Frei/Gebucht-Informationen in anderen Outlook-Kalendern anzeigen können oder wenn andere Personen keine Frei/Gebucht-Informationen in Ihrem Kalender sehen können, verwenden Sie den [Microsoft-Support-und Wiederherstellungs-Assistenten für Microsoft 365](https://diagnostics.office.com/).
