# <a name="calendar-permissions"></a>Kalenderberechtigungen

Wenn Sie Probleme haben, auf Ihren Kalender oder andere Kalender zuzugreifen, verwenden Sie den [Microsoft-Support-und Wiederherstellungs-Assistenten für Office 365](https://diagnostics.office.com/).
  
Weitere Informationen zur Freigabe Ihres Kalenders finden Sie unter [Freigeben eines Outlook-Kalenders für andere Personen](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88.aspx).
