# <a name="cant-add-default-2010-approval-workflow"></a>Standard-2010-Genehmigungs Workflow kann nicht hinzugefügt werden

In einer Microsoft SharePoint-Websitesammlung können Sie einer Liste oder Bibliothek keinen Global wiederverwendbaren Workflow (beispielsweise "Genehmigungs SharePoint 2010") hinzufügen. Der Genehmigungsworkflow fehlt möglicherweise. Um dieses Problem zu beheben, lesen Sie den Abschnitt [kann keinen Global wiederverwendbaren Workflow zu einer SharePoint-Liste oder-Bibliothek hinzufügen](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Weitere Informationen zu Workflow Genehmigungen finden Sie unter [alles zu Genehmigungsworkflows]https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742)(. 
 
Erfahren Sie mehr über [moderne Genehmigungen in Flow]https://flow.microsoft.com/blog/introducing-modern-approvals)(.
