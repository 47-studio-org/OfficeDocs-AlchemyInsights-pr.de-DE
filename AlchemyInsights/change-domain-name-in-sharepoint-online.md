# <a name="change-domain-name-in-sharepoint-online"></a>Ändern Sie Domänennamen in SharePoint Online

Leider ist es nicht möglich, den SharePoint Online Domänennamen für Ihre Organisation in Office 365 zu ändern. Wenn der Name Ihrer Organisation von "Contoso" in "Fourth Coffee" geändert wird, können nicht Sie beispielsweise contoso.sharepoint.com in fourthcoffee.sharepoint.com ändern.
  
Wenn das Domain Name fourthcoffee.sharepoint.com verwenden möchten, müssen Sie ein neues Office 365-Abonnement erwerben und verschieben Sie alle e-Mails, Dateien und alle anderen Daten, die Sie für das neue Abonnement beibehalten möchten.
  
 **Hinweis**: beispielsweise zum Ändern des Namens einer Website von https://contoso.sharepoint.com/sites/sample1 , https://contoso.sharepoint.com/sites/sample2, eine neue Website erstellen und anschließend alle Inhalte von der alten in die neue verschieben. 
  
Informationen zum Verschieben von Elementen in Dokumentbibliotheken finden Sie unter [verschieben oder Kopieren von Elementen aus einer SharePoint-Dokumentbibliothek](https://go.microsoft.com/fwlink/?linkid=2025831).
