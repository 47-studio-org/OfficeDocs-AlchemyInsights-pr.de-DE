# <a name="change-your-account-picture"></a>Ändern des Konto Bilds

1. Wechseln Sie zu [Ihrem Microsoft-Konto](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Wählen Sie **Bild ändern**aus. 
    
3. Wählen Sie **Neues Bild**aus. 
    
4. Wählen Sie das Bild aus, das Sie verwenden möchten, und wählen Sie **Öffnen**aus. 
    
5. Ziehen Sie das Bildfeld, oder ändern Sie die Größe, und wählen Sie **Speichern**aus. 
    
Sie können bis zu 24 Stunden dauern, bis Änderungen am Profilfoto oder-Name wirksam werden.
