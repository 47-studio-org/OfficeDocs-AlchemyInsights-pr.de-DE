# <a name="change-your-profile-name"></a>Ändern des Profilnamens

1. Wechseln Sie zu [Ihrem Microsoft-Konto](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Wählen Sie **Name bearbeiten**aus. 
    
3. Geben Sie einen vor-und Nachnamen ein, und wählen Sie **Speichern**aus. 
    
Sie können bis zu 24 Stunden dauern, bis Änderungen an Ihrem Profilnamen wirksam werden.
  
Weitere Informationen finden Sie unter [Ändern Ihres Profil Fotos oder namens in Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
