# <a name="changing-public-folder-permissions"></a>Ändern der Berechtigungen für Öffentliche Ordner

Berechtigungen für Öffentliche Ordner können von Benutzern und Administratoren in Outlook geändert werden. Administratoren können auch Berechtigungen aus dem Exchange-Verwaltungskonsole (EAC) steuern, indem Sie folgende Schritte ausführen:
  
1. Navigieren Sie im O365-Portal im linken Navigationsbereich zu **Admin Centers** \> **Exchange**.
    
2. Navigieren Sie zu **Öffentliche Ordner**.
    
3. Von dort aus können Sie Berechtigungen für einzelne Öffentliche Ordner ändern, indem Sie Berechtigungen Sicherheitsgruppen zuweisen. Damit ein Endbenutzerberechtigungen für Öffentliche Ordner ändern kann, muss der Benutzer Besitzerrechte für den Ordner besitzen.
