# <a name="check-teams-service-status"></a>Überprüfen des Teams-Dienststatus

Um den aktuellen Status des Microsoft Teams-Diensts zu überprüfen, wechseln Sie zum Dienststatus-Dashboard im Microsoft 365 Admin Center.

Weitere Informationen finden Sie unter [Überprüfen des Microsoft 365-Dienststatus](https://docs.microsoft.com/office365/enterprise/view-service-health).
