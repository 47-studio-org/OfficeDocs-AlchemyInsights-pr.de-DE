# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Konvertieren eines Benutzerpostfachs in ein freigegebenes Postfach

Wenn Sie dem Postfach eines Benutzers auf ein freigegebenes Postfach konvertieren, werden alle vorhandenen e-Mails und Kalender beibehalten. Nur jetzt ist es in einem freigegebenen Postfach sich mehrere Personen anstelle einer Person darauf zugreifen können. Zu einem späteren Zeitpunkt können Sie ein freigegebenes Postfach zurück in ein Benutzerpostfach (privat) konvertieren.
