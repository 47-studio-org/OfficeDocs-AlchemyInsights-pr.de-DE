# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Konvertieren eines Benutzerpostfachs in ein Raum- oder Gerätepostfach

Sie können das Postfach eines Benutzers mithilfe Exchange Online PowerShell in ein Raum-oder Gerätepostfach konvertieren. Weitere Informationen finden Sie unter [Konvertieren eines Postfachs](https://go.microsoft.com/fwlink/p/?LinkId=832875) .
