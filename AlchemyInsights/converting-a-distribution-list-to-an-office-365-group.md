# <a name="convert-a-distribution-group-to-an-office-365-group"></a>Konvertieren einer Verteilergruppe in eine Office 365 Gruppe

Sie können Verteilergruppen in Office 365 Gruppen umwandeln, wenn Sie bestimmte Kriterien erfüllen. Informationen zum Konvertieren einer Gruppe oder beheben von Konvertierungsproblemen finden Sie unter [Upgrade von Verteilerlisten auf Office 365 Gruppen in Outlook](https://support.office.com/article/787D7A75-E201-46F3-A242-F698162FF09F).
