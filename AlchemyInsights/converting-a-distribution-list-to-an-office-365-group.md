# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>Konvertieren einer Verteilergruppe in eine Microsoft 365-Gruppe

Sie können Verteilergruppen in Microsoft 365-Gruppen umwandeln, wenn Sie bestimmte Kriterien erfüllen. Informationen zum Konvertieren einer Gruppe oder beheben von Konvertierungsproblemen finden Sie unter [Upgrade von Verteilerlisten auf Microsoft 365-Gruppen in Outlook](https://docs.microsoft.com/office365/admin/manage/upgrade-distribution-lists).
