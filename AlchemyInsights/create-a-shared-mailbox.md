# <a name="create-a-shared-mailbox"></a>Erstellen eines freigegebenen Postfachs

Ein freigegebenes Postfach ermöglicht es mehreren Personen, e-Mails für einen Alias wie **Sales@Contoso.com**zu senden und zu empfangen, ohne eine andere Lizenz zuzuweisen.
  
Weitere Informationen finden Sie unter [Erstellen eines freigegebenen Postfachs](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) .
