# <a name="create-a-sharepoint-site"></a>Erstellen einer SharePoint-Website

Weitere Informationen finden Sie unter [Verwalten von Websites im neuen SharePoint Admin Center](https://docs.microsoft.com/sharepoint/manage-site-creation ) für Website Erstellungsoptionen. Wählen Sie diese Option aus, um eine [Teamwebsite](https://support.office.com/article/create-a-team-site-in-sharepoint-ef10c1e7-15f3-42a3-98aa-b5972711777d?ui=en-US&amp;rs=en-US&amp;ad=US) (mit der eine Office 365-Gruppe erstellt wird) oder eine [Kommunikationswebsite](https://support.office.com/article/7fb44b20-a72f-4d2c-9173-fc8f59ba50eb)zu erstellen. Um eine [klassische Website](https://docs.microsoft.com/sharepoint/manage-sites-in-new-admin-center#create-a-site)oder eine neue Teamwebsite zu erstellen, die keine Office 365-Gruppe enthält, klicken Sie auf **Weitere Optionen**. 
  
Tipps
- *Sie können keine Website mit derselben URL einer vorhandenen Website erstellen. Wenn Sie eine Website gelöscht haben und die URL erneut verwenden möchten, ist es möglich, dass die gelöschte Website weiterhin unter **Gelöschte Websites**vorhanden ist. Informationen zum Verwalten gelöschter Websites finden Sie unter [Löschen einer Website](https://docs.microsoft.com/sharepoint/manage-sites-in-new-admin-center#delete-a-site). Wenn Sie eine Website vollständig mit PowerShell entfernen möchten, finden Sie weitere Informationen unter dem Cmdlet [Remove-SPSite](https://docs.microsoft.com/sharepoint/manage-sites-in-new-admin-center#delete-a-site) .*
- *Einige Benutzer können möglicherweise keine Website erstellen. Weitere Informationen finden Sie unter [Manage Site Creation in SharePoint Online](https://docs.microsoft.com/sharepoint/manage-site-creation).*
- *Es ist möglich, dass die Website länger als erwartet nicht mehr **erstellt** wird. Wenn seit dem ersten Auftreten dieses Problems mehr als 24 Stunden vergangen sind, protokollieren Sie ein Support Ticket. In vielen Fällen arbeiten wir bereits an einer Lösung. Bitte geben Sie uns mindestens 24 Stunden Zeit, um eine Lösung abzuschließen.*
