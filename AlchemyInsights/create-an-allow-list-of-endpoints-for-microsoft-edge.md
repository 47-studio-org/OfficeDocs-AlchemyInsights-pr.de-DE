# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Erstellen einer Liste mit zulässigen Endpunkten für Microsoft Edge

Damit Ihnen das komplette Array von Microsoft Edge-Features zur Verfügung steht, muss es mit dem Internet verbunden sein. Um den Browser über Firewalls und andere Sicherheitsmechanismen im Internet kommunizieren zu lassen, müssen Sie außerdem eine Gruppe von Domänen-URLs zur [Zulassungsliste der Endpunkte](https://go.microsoft.com/fwlink/?linkid=2135054)hinzufügen.
