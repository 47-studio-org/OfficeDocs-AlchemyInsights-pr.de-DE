# <a name="data-location"></a>Datenspeicherort

Sie können den Standort Ihres Mandanten im Admin Center anzeigen, indem Sie zur Seite [ **Organisationsprofil** ](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) navigieren und **Datenspeicherort**auswählen.

Informationen zu Anwendungen, für die Sie nicht angemeldet sind, finden Sie unter [wo Ihre Kundendaten von Microsoft 365 gespeichert sind](https://docs.microsoft.com/office365/enterprise/o365-data-locations).
