# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Berechtigungen für Delegierte Postfächer in einer Hybridumgebung

Je nach der Version von Exchange, die Sie in Ihrer lokalen Organisation installiert haben, müssen Sie möglicherweise einige zusätzliche Schritte ausführen, um Delegierte Postfachberechtigungen zwischen Ihrer lokalen Organisation und Exchange Online in Office 365 einzurichten. Weitere Informationen zum Einrichten von Delegierten Berechtigungen finden Sie unter [Configure Exchange to Support Delegated Mailbox Permissions in a Hybrid Deployment](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
