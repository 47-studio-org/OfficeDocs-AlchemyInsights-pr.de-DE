# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Postfachberechtigungen in einer hybridumgebung

Je nach der Version von Exchange, die Sie in Ihrer lokalen Organisation installiert haben, müssen Sie möglicherweise einige zusätzliche Schritte zum Einrichten der delegierten Postfachberechtigungen zwischen Ihrer lokalen Organisation und Exchange Online in Office 365. Weitere Informationen dazu, wie Sie von delegierten Berechtigungen festlegen betrachten Sie [Exchange zur Unterstützung von delegierten Berechtigungen in einer hybridbereitstellung konfigurieren](https://technet.microsoft.com/en-us/library/mt784505%28v=exchg.150%29.aspx).
