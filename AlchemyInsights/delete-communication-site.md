# <a name="delete-a-communication-site"></a>Löschen einer Kommunikationswebsite

1. Wechseln Sie zur Website.
    
2. Klicken Sie oben rechts auf das Symbol "Einstellungen".
    
3. Klicken Sie auf **Websiteeinstellungen**. Klicken Sie unter **Website Informationen**auf **diese Website löschen**, und klicken Sie dann auf **Löschen**.
