# <a name="delete-a-sharepoint-site-collection"></a>Löschen einer SharePoint-Websitesammlung

1. Wählen Sie das Kontrollkästchen neben der Websitesammlung im klassischen SharePoint Administrationscenter, klicken Sie auf der Seite Site Collections.
    
2. Wählen Sie auf der Registerkarte Websitesammlungen **Löschen**.
