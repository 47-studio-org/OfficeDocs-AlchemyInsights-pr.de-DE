# <a name="delete-a-classic-sharepoint-team-site-or-subsite"></a>Löschen einer klassischen SharePoint-Teamwebsite oder-Unterwebsite

1. Wechseln Sie zur Website.
    
2. Klicken Sie oben rechts auf das Symbol " **Einstellungen** ". 
    
3. Klicken Sie auf **Websiteeinstellungen**. Klicken Sie unter **Websiteaktionen**auf **Website löschen**, und klicken Sie dann auf **Löschen**.
