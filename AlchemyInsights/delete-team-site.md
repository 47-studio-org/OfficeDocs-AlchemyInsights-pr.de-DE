# <a name="delete-a-sharepoint-team-site"></a>Löschen einer SharePoint-Teamwebsite

1. Wechseln Sie zur Website.
    
2. Klicken Sie oben rechts auf das Symbol " **Einstellungen** ". 
    
3. Klicken Sie auf **Website** Einstellungen. Klicken Sie unter **Website Informationen** oder **Websiteaktionen**auf **diese Website löschen**, und klicken Sie dann auf **Löschen**.
