# <a name="deleting-users"></a>Löschen von Benutzern

Wenn Sie sofort Benutzerzugriff und müssen ihre Informationen blockieren möchten, nehmen Sie finden Sie unter [ein Mitarbeiters verlassen haben und den Zugriff auf Daten eine e-Mail blockieren müssen?](https://support.office.com/client/ba665d35-f4af-4bd0-b52d-841df7454d4b)
  
Wenn Sie einfach den Benutzer löschen möchten, und sind nicht betroffenen mit ihren Daten, finden Sie unter [Löschen eines Benutzers aus Ihrer Organisation](https://support.office.com/article/d5155593-3bac-4d8d-9d8b-f4513a81479e). Dadurch werden sofort die Lizenz freigegeben.
