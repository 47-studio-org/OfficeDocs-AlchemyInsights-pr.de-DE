# <a name="deleting-users"></a>Löschen von Benutzern

Wenn Sie den Benutzer Zugriff sofort blockieren und seine Informationen behalten möchten, lesen Sie bitte [einen Mitarbeiter verlassen und müssen den Zugriff auf Daten per e-Mail blockieren.](https://support.office.com/client/ba665d35-f4af-4bd0-b52d-841df7454d4b)
  
Wenn Sie den Benutzer lediglich löschen möchten und keine Daten betreffen, lesen Sie bitte [Löschen eines Benutzers aus Ihrer Organisation](https://support.office.com/article/d5155593-3bac-4d8d-9d8b-f4513a81479e). Dadurch wird die Lizenz sofort freigegeben.
