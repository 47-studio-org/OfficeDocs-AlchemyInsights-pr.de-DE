# <a name="enable-seamless-sso"></a>Nahtloses SSO aktivieren

Azure Active Directory (Azure AD) nahtloses einmaliges Anmelden (Seamless SSO) automatisch bei Benutzern, wenn Sie sich auf Ihren Unternehmensdesktops befinden, die mit Ihrem Unternehmensnetzwerk verbunden sind.
  
[Befolgen Sie die Schritte in diesem Artikel, um die nahtlose SSO zu aktivieren](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
