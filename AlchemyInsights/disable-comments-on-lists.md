# <a name="disable-comments-on-lists"></a>Kommentare in Listen deaktivieren

Administratoren können das Feature auf Organisationsebene deaktivieren, indem Sie den **CommentsOnListItemsDisabled** -Parameter im PowerShell **-Cmdlet "Sets-SPOTenant** " ändern.

Es ist derzeit nicht möglich, das kommentieren auf Website-oder Listenebene zu deaktivieren. Wir hoffen, diese Steuerelemente in einem späteren Update zu haben, wahrscheinlich im ersten Quartal 2021.
