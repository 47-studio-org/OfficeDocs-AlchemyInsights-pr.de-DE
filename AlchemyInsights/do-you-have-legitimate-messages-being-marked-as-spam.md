# <a name="do-you-have-legitimate-messages-being-marked-as-spam"></a>Haben Sie legitime Nachrichten als Spam markiert wird?


Versuchen Sie die oben genannten Diagnose oder [Leitfaden für die Problembehandlung bei gute Nachrichten als Spam markiert wird](https://docs.microsoft.com/office365/securitycompliance/prevent-email-from-being-marked-as-spam-0).
  
Die beste Möglichkeit zum Melden dieser Art von Problemen werden empfohlen:
  
1. [Aktivieren Sie das reporting-add-in](https://docs.microsoft.com/office365/securitycompliance/enable-the-report-message-add-in) für die Benutzer. 
    
2. Haben Sie der Benutzer [Bericht die Nachricht als keine Junk](https://support.office.com/article/use-the-report-message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2?ui=en-US&amp;rs=en-US&amp;ad=US).
    
Wenn Sie eine Support-Ticket öffnen möchten, schließen die vollständigen e-Mail-Kopfzeilen aus mindestens **eine gesamte ursprüngliche Nachricht** unten.
