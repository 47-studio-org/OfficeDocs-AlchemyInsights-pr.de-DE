# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>Vollständiger Name wird von Unified Inteface-apps nicht unterstützt

Aktionen oder Bedingungen, die ein FullName-Feld ( **Vollständiger Name** ) verwenden, werden in apps basierend auf der einheitlichen Schnittstelle nicht unterstützt. Alternativ können Sie Aktionen oder Bedingungen mit Feldern **Vorname** **(FirstName) und Nachname** (LastName) verwenden.
