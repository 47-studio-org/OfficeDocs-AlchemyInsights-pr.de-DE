# <a name="edit-a-read-only-file"></a>Bearbeiten einer schreibgeschützten Datei

Wenn Sie in Dokumentbibliotheken auf den Namen eines Dokuments klicken, wird das Office-Programm möglicherweise unerwarteterweise als schreibgeschützte Datei geöffnet. Klicken Sie zum Öffnen des Bearbeitungsmodus auf **Öffnen** (die drei Punkte), und klicken Sie dann auf **Bearbeiten.**
