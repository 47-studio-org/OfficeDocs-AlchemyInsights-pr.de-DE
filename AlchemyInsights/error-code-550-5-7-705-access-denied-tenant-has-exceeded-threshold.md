# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 Zugriff verweigert, Mandanten Schwellenwert überschritten hat

Diese Meldung tritt auf, wenn zu viele Mail Spam oder Massen von Office 365-Organisation gesendet wurde. Wenn dieser Block entfernen möchten, müssen Sie verstehen, und erklären Sie die Ursache für den Support-Mitarbeiter als auch das zugrunde liegende Problem zu beheben.
