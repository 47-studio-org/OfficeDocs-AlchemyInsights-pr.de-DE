# <a name="what-is-autosave"></a>Was ist Autosave?

Autosave ist ein Feature, das in Excel, Word und PowerPoint für Office 365 Abonnenten zur Verfügung steht, mit denen die Datei während der Arbeit automatisch alle paar Sekunden gespeichert wird. 

Informationen zu häufig gestellten Fragen finden Sie unter [Was ist Autosave?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
