# <a name="export-user-account-information"></a>Exportieren von Benutzerkontoinformationen

Um Benutzerkontoinformationen im Microsoft 365 Admin Center zu exportieren, einschließlich Informationen zu Lizenzen, Kennwortrichtlinien, gesperrten Konten, AAD-Synchronisierungsstatus und mehr, wechseln Sie zur Seite **Benutzer** > [aktive Benutzer](https://go.microsoft.com/fwlink/p/?linkid=834822) , und wählen Sie dann **Benutzer exportieren**aus.
