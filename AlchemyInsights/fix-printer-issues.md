# <a name="troubleshoot-your-printer"></a>Problembehandlung beim Drucker

Wenn Sie Druckerprobleme haben, versuchen Sie zunächst, einen **Drucker Stromzyklus**auszuführen. Führen Sie die folgenden Schritte aus:

1. Schalten Sie den Drucker aus, und ziehen Sie das Netzkabel ab.
2. Warten Sie 30 Sekunden.
3. Schließen Sie den Drucker wieder an das Stromnetz an, und schalten Sie ihn ein.

Wenn Ihr Problem dadurch nicht behoben wurde, führen Sie die Drucker-Problembehandlung aus, indem Sie zum Thema **Start**  >  **Settings**  >  **Update & Security**  >  **Troubleshooting**wechseln. Wählen Sie **Drucker**aus, und wählen Sie dann **Problembehandlung ausführen**aus.
