# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Unterstützung für die Outlook Mobile-App

1. Öffnen Sie in der Outlook Mobile-App das **Menü**.
    
2. Wählen Sie das Fragezeichen aus, um den Bildschirm **Hilfe &amp; Feedback** zu öffnen. 
    
3. Tippen Sie auf **Support kontaktieren** , und erläutern Sie Ihr Problem. 
    
4. Tippen Sie auf **senden** , um eine Benachrichtigung in der APP zu erhalten, wenn Sie von einem Supportmitarbeiter kontaktiert werden. 
    
Ausführliche Anweisungen finden Sie unter [Get in-App Help for Outlook for Ios and Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
