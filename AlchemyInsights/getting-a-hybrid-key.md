# <a name="getting-an-exchange-hybrid-key"></a>Aufrufen eines Exchange-Hybrid Schlüssels

Um eine Verbindung zwischen Ihrer lokalen Organisation und Exchange Online in Office 365 herzustellen, müssen Sie mindestens einen Exchange-Server in Ihrer lokalen Organisation einrichten. Dieser Server dient zum Erstellen einer hybridbereitstellung, die Ihre lokale Exchange-Organisation mit Exchange Online verbindet. Um einen Hybrid Schlüssel für diesen Server zu erhalten, führen Sie die Schritte unter [Exchange Hybrid Product Key Distribution Wizard](https://aka.ms/hybridkey)aus.
