# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Fehler bei Deaktivierung eines Kontos in Outlook im Internet?

Wenn Sie die Fehlermeldung erhalten, mit `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` der **Ihr Konto deaktiviert wurde** , wenn Sie versuchen, Outlook im Internet zu öffnen (früher bekannt als Outlook Web App), hat ihr Organisationsadministrator möglicherweise Ihren Zugriff auf Outlook im Internet deaktiviert. Der Administrator kann diesen Fehler beheben, indem Sie die Schritte im Thema [Aktivieren oder deaktivieren Outlook Web App für ein Postfach](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)ausführen.
