# <a name="activating-windows-10-subscriptions"></a>Aktivieren von Windows 10-Abonnements

Microsoft 365 Business, Microsoft 365 Education, Microsoft 365 und Enterprise bieten digitale Lizenzen für Windows 10 als Teil des Abonnements. Damit das Abonnement aktiviert werden kann, muss das Gerät bereits Windows 10 pro aktiviert haben (1703 oder höher). Eine vollständige Liste der Anforderungen für die Aktivierung eines Windows 10-Abonnements finden Sie unter [Windows 10 Subscription Activation Requirements](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
