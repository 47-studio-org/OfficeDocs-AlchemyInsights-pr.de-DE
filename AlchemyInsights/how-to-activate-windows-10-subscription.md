# <a name="activating-windows-10-subscriptions"></a>Aktivieren von Windows 10-Abonnements

Microsoft 365 Business Premium, Microsoft 365 Education, Microsoft 365 und Enterprise bieten digitale Lizenzen für Windows 10 als Teil des Abonnements. Um das Abonnement zu aktivieren, muss Windows 10 Pro (1703 oder höher) bereits auf Ihrem Gerät aktiviert sein. Eine vollständige Liste der Anforderungen für die Aktivierung eines Windows 10-Abonnements finden Sie unter [Windows 10 Subscription Activation Requirements](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
