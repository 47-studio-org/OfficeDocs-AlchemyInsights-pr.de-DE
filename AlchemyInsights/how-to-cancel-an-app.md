# <a name="how-to-cancel-software-as-a-service-apps"></a>Abbrechen von Software-as-a-Service-apps 

1. Klicken Sie im Microsoft 365 Admin Center auf der Seite **Abrechnungs** > [Produkte #a0 Dienste](https://go.microsoft.com/fwlink/p/?linkid=842054) auf die Registerkarte **apps** .

2. Suchen Sie die APP, die Sie verwalten möchten, und wählen Sie Sie aus.

3. Wählen Sie unter **Einstellungen #a0 Aktionen**die Option **Abonnement kündigen**aus.
