# <a name="how-to-close-your-desktop-analytics-account"></a>So schließen Sie Ihr Desktop Analytics-Konto

Wenn Sie Desktop Analyse in Ihrer Umgebung einrichten und dann entscheiden, dass Sie Sie entfernen möchten, verwenden Sie den Prozess im folgenden Link:

[So schließen Sie Ihr Konto](https://docs.microsoft.com/sccm/desktop-analytics/account-close)

Der erste Schritt besteht darin, sich an den Microsoft-Support zu wenden. Öffnen Sie eine Supportanfrage, um Ihr Desktop Analytics-Konto zu schließen. Fahren Sie nicht mit weiteren Schritten fort, bis Sie eine Bestätigung erhalten, dass Microsoft Ihr Konto geschlossen hat.
