3. <span data-ttu-id="f59fa-p103">**Externe Benutzer in einer Unterhaltung Yammer in einer internen Gruppe** : Diese Option ermöglicht externen Teilnehmern zu einer bestimmten Unterhaltung hinzugefügt werden soll.
