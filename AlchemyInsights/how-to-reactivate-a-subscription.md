# <a name="how-to-reactivate-a-subscription"></a>Informationen zum Reaktivieren eines Abonnements

Wenn Ihr Abonnement abgelaufen ist, wurde abgebrochen oder von Microsoft deaktiviert wurde, können Sie es im Office 365 Administrationscenter reaktivieren.
  
1. Wechseln Sie im Admin Center zur Seite [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), oder wählen Sie **Abrechnung** \> **Abonnements** aus.
    
2. Wählen Sie das Abonnement, das Sie erneut aktivieren möchten.
    
3. Wählen Sie auf der rechten Seite auf die Schaltfläche **reaktivieren** . 
    
4. Sie werden aufgefordert, die Zahlungsdetails zu aktualisieren. Wenn alle Informationen korrekt sind, wählen Sie **Submit**aus.
    
5. Eine Bestätigung wird am unteren Rand des Bildschirms angezeigt. Wählen Sie **Schließen** um zur Seite **Abonnements** zurückzugeben. 
    
6. Wenn das Abonnement nicht aktiv angezeigt wird nach einigen Minuten aktualisieren Sie die Seite.
    
Weitere Informationen finden Sie unter: [Erneuern Office 365 für Unternehmen](https://support.office.com/article/8d83b530-f4ca-47f6-a666-e5791cbacc7e)
