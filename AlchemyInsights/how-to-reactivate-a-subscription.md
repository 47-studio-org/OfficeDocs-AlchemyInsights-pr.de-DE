# <a name="how-to-reactivate-a-subscription"></a>Reaktivieren eines Abonnements

Wenn Ihr Abonnement abgelaufen ist, gekündigt oder von Microsoft deaktiviert wurde, können Sie es im Microsoft 365 Admin Center reaktivieren.
  
1. Wechseln Sie im Admin Center zur Seite [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), oder wählen Sie **Abrechnung** \> **Abonnements** aus.
    
2. Wählen Sie das Abonnement aus, das Sie reaktivieren möchten.
    
3. Wählen Sie auf der rechten Seite die Schaltfläche **Reaktivieren** aus. 
    
4. Sie werden aufgefordert, die Zahlungsdetails zu aktualisieren. Wenn alle Informationen korrekt sind, wählen Sie **Absenden** aus.
    
5. Unten auf dem Bildschirm wird eine Bestätigung angezeigt. Wählen Sie **Schließen** aus, um zur Seite **Abonnements** zurückzukehren. 
    
6. Wenn das Abonnement nach ein paar Minuten nicht als "Aktiv" angezeigt wird, aktualisieren Sie die Seite.
    
Weitere Informationen finden Sie unter [Verlängern von Office 365 Business](https://support.office.com/article/8d83b530-f4ca-47f6-a666-e5791cbacc7e).
