# <a name="how-to-view-and-manage-apps"></a>Anzeigen und Verwalten von apps

Sie können Ihre apps im Microsoft 365 Admin Center anzeigen.

1. Navigieren Sie im Admin Center zur Seite **Abrechnung** > [Ihre Produkte](https://go.microsoft.com/fwlink/p/?linkid=842054).
2. Führen Sie auf der Registerkarte **Produkte** einen Bildlauf nach unten zum Abschnitt **apps** durch.
3. Um eine APP zu verwalten, wählen Sie diejenige aus, die Sie verwalten möchten.
