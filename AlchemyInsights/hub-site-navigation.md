# <a name="sharepoint-hub-site-navigation"></a>SharePoint-Hub-Websitenavigation

Am oberen Rand eines Hub-Standorts und der zugehörigen Standorte oberhalb der Websitenavigation befindet sich die Navigationsleiste Hub-Website. Wenn Sie der Hub-Websitebesitzer sind, können Sie die [Navigationsleiste](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav)anpassen. 

**Hinweis:** Es wird empfohlen, eine Kommunikationswebsite oder eine Teamwebsite auszuwählen, die die moderne Vorlage als Hub-Standort verwendet. Wenn Sie eine klassische Teamwebsite verwenden, wird die Hub-Navigation nur auf den modernen Seiten angezeigt, und die Hub-Standorteinstellungen werden nur auf der modernen Seite angezeigt.
