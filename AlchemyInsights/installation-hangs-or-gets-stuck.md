# <a name="installation-hangs-or-gets-stuck"></a>Installation reagiert nicht mehr oder bleibt hängen

Möglicherweise wird dieses Verhalten von Netzwerkkonnektivität, Antivirensoftware oder Windows Update-Einstellungen auf Ihrem Computer angezeigt. Hier sind einige Punkte, die Sie zum Beheben des Problems ausprobieren können.
  
- Starten Sie den Computer neu, und überprüfen Sie, ob die Installation von Office
    
- Eine drahtgebundene Verbindung verwenden
    
- Wenn Sie auf eine langsame Verbindung zum Internet haben, installieren Sie Office [mithilfe des Office-offline-Installers](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA.aspx)
    
- Antivirensoftware vorübergehend deaktivieren
    
- Ausführen von onlinereparatur von Control Panel
    
- Entfernen und Neuinstallieren von Office
    
[Office wird so installieren Sie lange dauert.](https://support.office.com/article/0f09f357-3fef-42a6-b8aa-cef4c6c44bdf?wt.mc_id=Alchemy_ClientDIA.aspx)
