# <a name="learn-about-external-calendar-sharing-options"></a>Informationen zu externen Kalenderfreigabe Optionen

Wenn Sie Besprechungen mit Personen außerhalb Ihrer Organisation planen müssen, können Sie mit Office 365 Kalender für "externe Benutzer" zur Verfügung stellen, die Frei/Gebucht-Zeiten anzeigen müssen, aber keine Benutzerkonten für Ihre Office 365 Umgebung haben. Weitere Informationen finden Sie unter [Freigeben von Kalendern für externe Benutzer](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
