# <a name="manage-recurring-billing"></a>Verwalten von wiederkehrenden Rechnungen

Zum Verwalten der wiederkehrenden Abrechnung können Sie die **wiederkehrende Abrechnung** ein-oder ausschalten.
  
1. Wechseln Sie im Admin Center zur Seite **Abrechnungs** \> [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Wählen Sie das Abonnement aus, für das Sie wiederkehrende Abrechnung verwalten möchten.

3. Wechseln Sie die Umschaltfläche für **wiederkehrende Fakturierung** **auf ein** oder **aus**.

Weitere Informationen finden Sie unter [Turn wiederkehrende Abrechnung aus oder ein](https://docs.microsoft.com/en-us/office365/admin/subscriptions-and-billing/renew-your-subscription?view=o365-worldwide#turn-recurring-billing-off-or-on).
