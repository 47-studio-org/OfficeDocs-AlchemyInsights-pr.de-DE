# <a name="manage-synchronized-user-accounts"></a>Synchronisierte Benutzerkonten verwalten

In der Regel verwenden Sie das Office 365 Administrationscenter, hinzufügen und Ändern von Benutzerkonten. Es kann nicht jedoch verwenden, wenn Directory-Synchronisierung oder SSO aktiviert sind. Verwalten von Benutzern mit Windows PowerShell stattdessen wie unter [Konfigurieren der Eigenschaften von Benutzerkonten mit Office 365 PowerShell](https://docs.microsoft.com/en-us/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell )beschrieben.
