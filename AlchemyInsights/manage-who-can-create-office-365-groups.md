# <a name="manage-who-can-create-office-365-groups"></a>Verwalten von Personen, die Office 365-Gruppen erstellen können

Standardmäßig können alle Benutzer in Ihrer Organisation O365 Gruppen erstellen.
  
Wenn Sie diese Einstellung ändern möchten, können Sie eine Sicherheitsgruppe mit PowerShell erstellen, die Erstellung von Gruppen auf eine Teilmenge der Benutzer beschränkt.
  
Führen Sie eine schrittweise Anleitung: [Benutzer, Office 365 Gruppen erstellen können verwalten](https://support.office.com/article/4c46c8cb-17d0-44b5-9776-005fced8e618)
