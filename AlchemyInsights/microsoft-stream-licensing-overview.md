# <a name="microsoft-stream-licensing-overview"></a>Übersicht über die Microsoft Stream-Lizenzierung

Microsoft Stream ist ein clouddienst, der auf Abonnementbasis pro Benutzer lizenziert ist. Es ist als Teil berechtigter Office 365-Abonnements verfügbar. Informationen zu den einzelnen Abonnementtypen finden Sie unter Office 365 Preise.

- [Übersicht über die Microsoft Stream-Lizenzierung](https://docs.microsoft.com/stream/license-overview)
