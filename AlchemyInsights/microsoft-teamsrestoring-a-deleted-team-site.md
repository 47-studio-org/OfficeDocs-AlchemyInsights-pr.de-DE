# <a name="restoring-a-deleted-team-in-microsoft-teams"></a>Wiederherstellen eines gelöschten Teams in Microsoft Teams

Haben Sie ein Team versehentlich gelöscht? Wenn Sie weniger als 30 Tage vergangen sind, seit Sie das Team gelöscht haben, haben Sie Glück; Sie können es wiederherstellen. Führen Sie dazu die folgenden Schritte aus: [Wiederherstellen eines gelöschten Teams](https://blogs.technet.microsoft.com/skypehybridguy/2017/07/23/restoring-a-deleted-team-in-microsoft-teams/)
