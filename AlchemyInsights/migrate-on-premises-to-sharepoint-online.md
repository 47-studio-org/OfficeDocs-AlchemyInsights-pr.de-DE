# <a name="migrate-on-premises-to-sharepoint-online"></a>Migrieren von lokalen zu SharePoint Online

Um aus einer lokalen SharePoint-Serverfarm mit SharePoint Online zu migrieren, können Sie das Migrationstool der SharePoint verwenden. Weitere Informationen über das Tool finden Sie unter [Migrieren zu SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)
