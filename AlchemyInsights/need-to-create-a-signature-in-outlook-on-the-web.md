# <a name="need-to-create-an-email-signature"></a>Sie müssen eine e-Mail-Signatur erstellen?

Wenn Sie über ein Outlook-Konto oder ein Outlook im Internet-Konto (Office 365 work or School) verfügen, können Sie eine Signatur erstellen und Sie automatisch allen ausgehenden Nachrichten hinzufügen oder selektiv zu bestimmten ausgehenden Nachrichten hinzufügen. Beachten Sie, dass Ihre e-Mail-Signatur in Outlook und Outlook im Internet unterschiedlich ist. Informationen zum Erstellen Ihrer e-Mail-Signatur finden Sie unter [Erstellen und Hinzufügen einer Signatur zu Nachrichten in Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) oder [Erstellen und Hinzufügen einer e-Mail-Signatur in Outlook im Internet](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
