# <a name="add-your-other-email-accounts-to-outlookcom"></a>Hinzufügen Ihrer anderen e-Mail-Konten zu Outlook.com

Wenn Sie Outlook.com weitere e-Mail-Konten hinzufügen, können Sie Outlook.com verwenden, um e-Mail-Nachrichten von diesen Konten zu senden und zu lesen, ohne Outlook.com zu verlassen. Jedes Konto, das Sie Outlook.com hinzufügen, wird als verbundenes Konto bezeichnet.

Hinweis: die Synchronisierung von Ihrem verbundenen Konto mit Ihrem Outlook.com-Konto ist nur unidirektional. E-Mails, die von Ihrem verbundenen Konto mithilfe von Outlook.com gesendet werden, werden nicht in Ihrem verbundenen Konto gesendete Elemente angezeigt.

So fügen Sie ein verbundenes Konto hinzu

1. Öffnen Sie die Einstellungen für die [Synchronisierungs-e-Mail](https://go.microsoft.com/fwlink/?linkid=875264).
2. Wählen Sie unter **Verbundene Konten**die Option **Gmail** oder **andere e-Mail-Konten**aus.
3. Geben Sie auf der Seite Ihr Konto verbinden einen Anzeigenamen ein (der Name der Empfänger wird angezeigt, wenn Sie eine e-Mail-Nachricht von Ihnen erhalten) und die vollständige e-Mail-Adresse und das Kennwort für das e-Mail-Konto, das Sie mit Ihrem Outlook.com-Konto verbinden möchten.

Sie können e-Mails von einem beliebigen Konto aus senden, das mit Ihrem Outlook.com-Konto verbunden ist. Wenn Sie eine e-Mail verfassen, wählen Sie **aus** , wählen Sie die e-Mail-Adresse aus, die Sie verwenden möchten, oder [Aktualisieren Sie Ihre Standardadresse](https://go.microsoft.com/fwlink/?linkid=875264).

Weitere Informationen finden Sie unter [Hinzufügen Ihrer anderen e-Mail-Konten zu Outlook.com](https://support.office.com/article/c5224df4-5885-4e79-91ba-523aa743f0ba).

Wenn Sie eine neue e-Mail-Adresse erstellen möchten, die das gleiche Outlook.com-Konto verwendet, lesen Sie [Hinzufügen oder Entfernen eines e-Mail-Alias](https://support.office.com/article/459b1989-356d-40fa-a689-8f285b13f1f2).
