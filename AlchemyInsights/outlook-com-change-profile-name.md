# <a name="change-your-profile-name-in-outlookcom"></a>Ändern Ihres Profilnamens in Outlook.com

1. Wechseln [Sie zu Ihren Microsoft-Kontoinformationen](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Wählen Sie **Namen bearbeiten**aus.
3. Geben Sie vor-und Nachname ein, und wählen Sie **Speichern**aus.

Lassen Sie bitte bis zu 24 Stunden zu, bis Änderungen an Ihrem Profilnamen wirksam werden.
