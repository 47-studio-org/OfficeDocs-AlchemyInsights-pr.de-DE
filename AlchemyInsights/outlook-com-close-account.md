# <a name="close-your-outlookcom-account"></a>Schließen Ihres Outlook.com-Kontos

1. Wechseln Sie zur Seite [Konto schließen](https://go.microsoft.com/fwlink/p/?linkid=845493) .
2. Befolgen Sie die Anweisungen auf der Seite, um sicherzustellen, dass Ihr Konto geschlossen werden kann, und wählen Sie **weiter**aus.
3. Lesen Sie die Liste, und aktivieren Sie die Kontrollkästchen, um zu bestätigen, dass Sie die einzelnen Elemente gelesen haben.
4. Wählen Sie in der Dropdownliste **Wählen Sie einen Grund** aus den Grund aus, warum Sie das Konto schließen.
5. Wählen Sie **Konto für Schließung markieren**aus.

Weitere Informationen finden Sie unter [Schließen Ihres Outlook.com-Kontos](https://support.office.com/article/564b801e-2a47-4cb2-afa8-12ead3185038?wt.mc_id=Office_Outlook_com_Alchemy).
