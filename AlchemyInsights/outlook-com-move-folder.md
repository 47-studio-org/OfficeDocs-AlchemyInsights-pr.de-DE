# <a name="move-a-folder"></a>Ordner umlegen

Sie können Ordner unter **Favoriten** ziehen, um Sie neu anzuordnen.
  
Sie können Ordner nicht Unterordner **** neu anordnen oder einen Unterordner zu einem Ordner migrieren.

*Hinweis:* Wenn Sie einen Ordner zu einem Unterordner migrieren, erstellen Sie einen neuen Ordner, und kopieren Sie alle Ihre e-Mails aus dem Unterordner in den neuen Ordner.
  
Weitere Informationen finden Sie unter [Arbeiten mit Nachrichtenordnern in Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
