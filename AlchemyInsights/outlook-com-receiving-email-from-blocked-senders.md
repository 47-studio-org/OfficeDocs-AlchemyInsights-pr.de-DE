# <a name="receiving-email-from-blocked-senders"></a>Empfangen von e-Mails von blockierten Absendern

Wenn e-Mails von einem blockierten Absender weiterhin in Ihrem Posteingang angezeigt werden, wird der Absender möglicherweise seine tatsächliche e-Mail-Adresse ausblenden. So überprüfen Sie die tatsächliche e-Mail-Adresse eines Absenders:
  
1. Wählen Sie die drei Punkte oben im Nachrichtenfenster aus, und wählen Sie **Nachrichtenquelle anzeigen**aus.
2. Führen Sie im Fenster **Nachrichtenquelle** einen Bildlauf nach unten durch, bis Sie das Feld **von** finden.
3. Markieren Sie die in \< \> eingeschlossene e-Mail-Adresse und klicken Sie mit der rechten Maustaste zu **Kopieren**
4. Verwenden Sie die kopierte Adresse, um den Absender zu blockieren. Informationen zum Blockieren von Absendern finden Sie unter [kennzeichnen von e-Mails als Junk-e-Mail oder Blockieren von Absendern in Outlook.com](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4?wt.mc_id=Office_Outlook_com_Alchemy).

Weitere Informationen finden Sie unter [empfangen von e-Mails von blockierten Absendern in Outlook.com](https://support.office.com/article/265923a0-b52c-4157-92c8-370385215da1?wt.mc_id=Office_Outlook_com_Alchemy).
