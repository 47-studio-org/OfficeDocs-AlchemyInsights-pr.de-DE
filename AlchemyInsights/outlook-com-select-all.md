# <a name="how-to-select-all-in-outlookcom"></a>Vorgehensweise "Select all" in Outlook.com

Sie können nur alle auswählen, wenn alle Nachrichten angezeigt werden. Dadurch werden unbeabsichtigte Aktionen für Nachrichten verhindert, die nicht angezeigt werden.

**STRG + Klicken** wählt mehrere einzelne Nachrichten aus.

**UMSCHALT + Klicken** wählt alle Nachrichten zwischen der ersten und der letzten Nachricht aus, die Sie auswählen.
