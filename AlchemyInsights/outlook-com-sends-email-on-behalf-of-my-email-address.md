# <a name="outlookcom-sends-email-on-behalf-of-my-email-address"></a>Outlook.com sendet e-Mail "im Namen von" meine e-Mail-Adresse

E-Mails, die von einem verbundenen Konto gesendet werden, werden als gesendet "im Namen" Ihres Microsoft-Kontos angezeigt. Dies ist ein bekanntes Problem und wird in einer zukünftigen Version behoben. Bis dahin können Sie Ihre Absenderadresse so ändern, dass Sie mit Ihrem primären Microsoft-Konto Alias identisch ist.
  
Weitere Informationen finden Sie unter [Outlook.com e-Mail senden "im Namen von" meine e-Mail-Adresse](https://go.microsoft.com/fwlink/p/?linkid=2001600&amp;clcid=0x409).
