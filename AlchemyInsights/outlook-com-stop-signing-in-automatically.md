# <a name="how-to-stop-signing-in-automatically"></a>So beenden Sie die Anmeldung automatisch

Wenn Sie die Anmeldung bei Outlook.com automatisch beenden möchten, deaktivieren Sie das Kontrollkästchen für **angemeldet bleiben** auf der Seite, auf der Sie Ihr Kennwort eingeben.
  
Weitere Informationen finden Sie unter [Anmelden bei oder bei Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy).
