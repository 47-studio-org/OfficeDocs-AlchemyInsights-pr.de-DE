# <a name="turn-off-focused-inbox"></a>Deaktivieren des Posteingangs mit Fokus

1. Wählen Sie oben auf der Seite das Zahnradsymbol aus, um den Bereich "Einstellungen" zu öffnen.
2. Verschieben Sie im Bereich Einstellungen die Option **fokussierter** Posteingangs Schalter auf aus.

Weitere Informationen finden Sie unter [Turn Off Focused Inbox](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2?wt.mc_id=Office_Outlook_com_Alchemy#bkmk_outlookcom).
