# <a name="how-to-view-message-headers-or-message-source"></a>So zeigen Sie Nachrichtenkopfzeilen oder Nachrichtenquellen an

Wenn Sie die Kopfzeilen und andere Details in einer e-Mail anzeigen möchten, wählen Sie die drei Punkte in der oberen Ecke des Lesebereichs aus, und wählen Sie **Nachrichtenquelle anzeigen**aus.
