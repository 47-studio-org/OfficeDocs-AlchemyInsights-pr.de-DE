# <a name="import-contacts"></a>Kontakte importieren

Bevor Sie Kontakte in Outlook im Internet importieren, müssen Sie Sie aus Ihrem anderen e-Mail-Anbieter als CSV-Datei exportieren. Folgen Sie dann den Anweisungen unten.

1. Wählen Sie unten auf der Seite das Personensymbol aus, um zur Personenseite zu wechseln.
2. Wählen Sie auf der Symbolleiste die Option**Import Kontakte** **Verwalten** > aus.
3. Wählen Sie **Durchsuchen**aus, wählen Sie Ihre CSV-Datei aus, und wählen Sie dann **Öffnen**aus.
4. Wählen Sie **Importieren** aus.

Weitere Informationen finden Sie unter [Importieren von Kontakten in Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
