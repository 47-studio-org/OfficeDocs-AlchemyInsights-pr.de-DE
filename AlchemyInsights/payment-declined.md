# <a name="update-or-add-a-new-payment-method"></a>Aktualisieren oder Hinzufügen einer neuen Zahlungsmethode

1. Wechseln Sie zur Seite **Billing** > **Bills & Payments** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">Payment Methods</a> .
2. Wählen Sie **Zahlungsmethode hinzufügen** aus, oder wählen Sie die zu aktualisierende Karte oder das Bankkonto aus, und klicken Sie dann auf **Bearbeiten**.
3. Aktualisieren Sie Ihre Zahlungsmethoden Informationen (Name auf der Karte oder Bankverbindung, Rechnungsadresse, Telefonnummer oder Ablaufdatum), und wählen Sie dann **Speichern**aus.
4. Denken Sie daran, zu ändern, wie Sie Ihre Abonnements für diese neue Zahlungsmethode bezahlen. Anweisungen zur Vorgehensweise finden Sie im nächsten Abschnitt.

Wenn Sie eine neue Karte hinzugefügt haben und immer noch einen Kündigungs Bescheid auf der alten Kreditkarte erhalten, müssen Sie diese neue Karte möglicherweise dem Abonnement zuordnen.

1. Wechseln Sie zu **Abrechnungs** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Produkte & Dienste</a>
2. Wählen Sie das Abonnement aus, das Sie mit dem neuen Kreditkarten-oder Bank Konto bezahlen möchten. Wählen Sie im Abschnitt **Abrechnung** neben der Zahlungsmethode die Option **Bearbeiten**aus.
3. Wählen Sie neben der aktuellen Zahlungsmethode **ändern** aus.
4. Wählen Sie im Dropdownmenü eine alternative Zahlungsmethode aus, oder fügen Sie eine neue Zahlungsmethode hinzu.

[Hinzufügen, aktualisieren oder Entfernen Ihrer Zahlungsmethoden](https://go.microsoft.com/fwlink/?linkid=2118133)
