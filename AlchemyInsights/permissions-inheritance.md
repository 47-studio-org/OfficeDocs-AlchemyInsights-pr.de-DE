# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Funktionsweise der Vererbung von Berechtigungen in SharePoint

Standardmäßig werden Berechtigungen in SharePoint höhere oben in der Hierarchie geerbt. Also: eine Datei erbt die Berechtigungen aus dem Ordner, der die Berechtigungen aus der Bibliothek erbt, die ihre Berechtigungen von der Website erbt die erbt die Berechtigungen aus der Websitesammlung.
  
Info zum Entfernen von eindeutiger Berechtigungen und Vererbung wiederherstellen können finden Sie unter [Bearbeiten und Verwalten von Berechtigungen für eine Liste oder Bibliothek](https://go.microsoft.com/fwlink/?linkid=869946).
