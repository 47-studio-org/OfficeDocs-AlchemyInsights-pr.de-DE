# <a name="prepaid-renewal"></a>Voraus bezahlte Verlängerung

1. Wechseln Sie im Admin Center zur Seite [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), oder wählen Sie **Abrechnung** \> **Abonnements** aus.
    
2. Wählen Sie das Voraus bezahlte Abonnement aus, das Sie erweitern möchten, und wählen Sie dann **erweitern**aus.
    
3. Wählen Sie die Option **Product Key**aus. (Wenn es sich um einen Product Key handelte, der an einem Einzelhandelsstandort erworben wurde, haben Sie möglicherweise die Option, sich mit anderen Zahlungsmethoden zu erweitern.)
    
4. Geben Sie den Product Key ein.
    
5. Stellen Sie sicher, dass die Informationen richtig sind, und klicken Sie dann auf **weiter**.
    
6. Wählen Sie das Optionsfeld aus, um das Enddatum der vorhandenen Lizenzen zu verlängern, und klicken Sie dann auf **weiter**.
