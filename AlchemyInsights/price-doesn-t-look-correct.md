# <a name="price-doesnt-look-correct"></a>Der Preis sieht nicht richtig aus?

Wenn der Preis auf Ihrer Rechnung nicht richtig aussieht, kann dies auf eine Preisänderung bei der Verlängerung zurückzuführen sein. Wenn Sie nicht glauben, dass dies das Problem erläutert, verwenden Sie die Option "lassen Sie uns anrufen", um mit einem Supporttechniker zu sprechen.
