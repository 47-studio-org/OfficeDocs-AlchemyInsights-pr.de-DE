# <a name="proxyaddress-incorrect"></a>ProxyAddress falsch

Wenn ein Objekt mit Azure AD synchronisiert wird, werden die Werte, die im proxyAddresses-Attribut in Active Directory angegeben sind, mit Azure AD Regeln verglichen, und dann wird das proxyAddresses-Attribut in Azure AD aufgefüllt. Daher sind die Werte des proxyAddresses-Attributs für das Objekt in Active Directory möglicherweise nicht identisch mit den Werten des proxyAddresses-Attributs in Azure AD.
  
Weitere Informationen zum Auffüllen des proxyAddress finden Sie unter [wie das proxyAddress-Attribut in Azure AD aufgefüllt wird](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
