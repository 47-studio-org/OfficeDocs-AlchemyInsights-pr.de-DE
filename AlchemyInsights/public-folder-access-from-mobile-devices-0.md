# <a name="public-folder-access-from-mobile-devices"></a>Zugriff auf Öffentliche Ordner von mobilen Geräten

Auf Öffentliche Ordner kann von mobilen Geräten nicht zugegriffen werden.
  
Wenn Sie ähnliche Funktionen wie öffentliche Ordner haben und von mobilen Geräten darauf zugreifen möchten, finden Sie weitere Informationen zu [Microsoft 365-Gruppen](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) für eine bessere Alternative.
