# <a name="reactivate-a-subscription"></a>Reaktivieren eines Abonnements

Wenn Ihr Abonnement abläuft, von Microsoft deaktiviert wurde oder es in der Mitte eines Abonnementzeitraums storniert hat, können Sie es im Admin Center reaktivieren.
  
1. Wechseln Sie im Admin Center zur Seite **Abrechnungs** \> [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Wählen Sie das Abonnement aus, das Sie reaktivieren möchten, und klicken Sie dann auf **Reaktivieren**.

    Wenn reaktivieren nicht als **** verfügbare Aktion angezeigt wird, verwenden Sie die Option "Wir rufen Sie an", um mit einem Supporttechniker zu sprechen, um Ihr Abonnement zu reaktivieren.

3. Geben Sie Ihre Zahlungsdetails ein. Hier können Sie Ihre vorhandenen Zahlungsinformationen aktualisieren.

Verwenden Sie das neue Admin Center? Neue Schritte finden Sie unter [reaktivieren Ihres Abonnements](https://docs.microsoft.com/en-us/office365/admin/subscriptions-and-billing/reactivate-your-subscription).
