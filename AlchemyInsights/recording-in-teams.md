# <a name="recording-in-teams"></a>Aufzeichnen in Teams

Wenn Sie in Teams nicht aufzeichnen können oder wissen müssen, wie Sie in Teams aufzeichnen können, versuchen Sie es mithilfe eines der folgenden Artikel.

Die Teams-Funktionen für Besprechungsaufzeichnungen hängen von der Stream-Verfügbarkeit in der Region bzw. im Land ab.  Wenn Sie eine Besprechung nicht aufzeichnen können, überprüfen Sie, ob [Stream in Ihrem Land oder Ihrer Region verfügbar ist](https://docs.microsoft.com/stream/faq#which-regions-does-microsoft-stream-host-my-data-in).  Wenn Sie weitere Informationen erhalten möchten, lesen Sie den folgenden Nachrichtencenterbeitrag im Administratorportal:

- [Updated Feature: Microsoft Stream cloud recordings regional storage control](https://admin.microsoft.com/AdminPortal/Home#/MessageCenter?id=MC214327) (Aktualisiertes Feature: Microsoft Stream-Cloudaufzeichnungen – regionale Speicherkontrolle)

Zur Nutzung der Besprechungsaufzeichnung in Teams [müssen bestimmte Lizenzen und Richtlinien zugewiesen werden](https://docs.microsoft.com/microsoftteams/cloud-recording#prerequisites-for-teams-cloud-meeting-recording), beispielsweise Stream-Lizenzen zum Hochladen oder Herunterladen von Besprechungsaufzeichnungen in Microsoft Stream.

Wenn Sie wissen möchten, wie Sie das Aufzeichnungsfeature in Teams für Ihre Organisation aktivieren und konfigurieren können, einschließlich Steuerelemente zum Aktivieren oder Deaktivieren von Besprechungsaufzeichnungen für bestimmte Benutzer, lesen Sie [Aufzeichnung einer Teams-Cloudbesprechung](https://docs.microsoft.com/microsoftteams/cloud-recording).

Nachdem Sie sich vergewissert haben, dass alle Anforderungen, einschließlich Lizenzierung, Richtlinien und regionaler Verfügbarkeit von Stream, erfüllt sind, lesen Sie [Aufzeichnen einer Besprechung in Teams](https://support.office.com/article/34dfbe7f-b07d-4a27-b4c6-de62f1348c24).
