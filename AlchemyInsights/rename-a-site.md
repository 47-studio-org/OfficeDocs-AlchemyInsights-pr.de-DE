# <a name="rename-a-site"></a>Umbenennen einer Website

So ändern Sie den Namen einer Website, beispielsweise von https://contoso.sharepoint.com/sites/sample1 , https://contoso.sharepoint.com/sites/sample2, müssen Sie eine neue Website erstellen und anschließend alle Inhalte von der alten in die neue verschieben. Informationen zum Verschieben von Elementen in Dokumentbibliotheken finden Sie unter [verschieben oder Kopieren von Elementen aus einer SharePoint-Dokumentbibliothek](https://go.microsoft.com/fwlink/?Linkid=2018691).
  
Hinweis: Wenn Sie den Domänennamen (d. h., die Contoso-Teil der URL oben) in einem SharePoint Online-URL ändern möchten, ist das nicht möglich. Finden Sie [Ihre Domänen ändern](https://go.microsoft.com/fwlink/?Linkid=2018696).
