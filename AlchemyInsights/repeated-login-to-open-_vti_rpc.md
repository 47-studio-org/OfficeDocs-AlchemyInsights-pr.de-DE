# <a name="repeated-login-to-open-_vti_rpc"></a>Wiederholte Anmeldung zum Öffnen _vti_rpc

Wenn Sie mit Office 2016 auf OneDrive-Dateien arbeiten, wird möglicherweise die folgende Meldung angezeigt: "teilen Sie uns das Konto mit, das Sie zum Öffnen von _vti_rpc verwenden möchten." Dieses Problem wurde in Version 1708 (16.0.8431.2094) behoben. Um dies zu beheben, aktualisieren Sie auf die neueste Version von Office. Wählen **File** \> Sie Update **Update Options** Optionen \> für Datei **Konten** \> **Jetzt aktualisieren**aus.
