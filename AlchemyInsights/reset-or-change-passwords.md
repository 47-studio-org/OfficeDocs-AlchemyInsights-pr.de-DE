# <a name="reset-or-change-office-365-business-passwords"></a>Zurücksetzen oder ändern Office 365 Geschäfts Kennwörter

Ein Benutzer kann sein eigenes Kennwort von " [mein Konto", &amp; "Sicherheit (Privacy](https://portal.office.com/account/#security))", ändern.
  
Wenn der Benutzer sein Kennwort vergessen hat, kann ein Administrator das Kennwort zurücksetzen, indem er zu **Benutzer** > [aktiven Benutzern](https://portal.office.com/adminportal/home#/users)wechseln, die Benutzer auswählen und **Kennwort zurücksetzen**auswählen.
  
[Klicken Sie hier, um weitere Informationen zu erhalten](https://support.office.com/article/admins-reset-office-365-business-passwords-7a5d073b-7fae-4aa5-8f96-9ecd041aba9c)oder Benutzer ihre vergessenen Kennwörter zurücksetzen zu lassen.
