# <a name="self-service-password-reset-sspr"></a>Zurücksetzen von Kennwörtern durch den Benutzer (Self-Service Password Reset, SSPR)

- Klicken Sie hier, um [Benutzern das Zurücksetzen ihrer eigenen Kennwörter zu gestatten](https://admin.microsoft.com/Adminportal/Home#/featureexplorer/security/Sspr).  SSPR ist standardmäßig für alle Administratoren aktiviert.

- Klicken Sie hier, um [Ihre SSPR-Kontaktinformationen zu registrieren](https://go.microsoft.com/fwlink/?linkid=849451).

Weitere Informationen:

Basiseinrichtung von SSPR – [Benutzern das Zurücksetzen ihrer eigenen Kennwörter gestatten](https://docs.microsoft.com/microsoft-365/admin/add-users/let-users-reset-passwords?view=o365-worldwide)

[SSPR – Häufig gestellte Fragen](https://docs.microsoft.com/azure/active-directory/authentication/active-directory-passwords-faq)

[SSPR – Problembehandlung](https://docs.microsoft.com/azure/active-directory/authentication/active-directory-passwords-troubleshoot)
