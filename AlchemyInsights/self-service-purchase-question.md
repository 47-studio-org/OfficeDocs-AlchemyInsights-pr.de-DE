# <a name="self-service-purchase"></a>Self-Service-Kauf

**Welche Funktionen hat ein Administrator für Self-Service-Käufe?**

Administratoren können alle Self-Service-Käufe anzeigen, die in Ihrer Organisation im Microsoft 365 Admin Center vorgenommen wurden. Sie können das Produkt, den Käufer Namen, die erworbenen Abonnements, das Ablaufdatum, den Bestellverlauf, den Kaufpreis und die zugewiesenen Benutzer für jeden Self-Service-Kauf anzeigen.  Wenn es für Ihre Organisation erforderlich ist, können Administratoren den Self-Service-Einkauf per PowerShell auf Produktbasis deaktivieren.  Administratoren haben die gleiche Datenverwaltung und Zugriffsrichtlinien für Produkte, die über Self-Service Purchase oder zentral erworben wurden.
