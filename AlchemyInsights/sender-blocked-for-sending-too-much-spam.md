# <a name="account-is-restricted-for-sending-too-much-spam"></a>Konto wegen des Versands von zu viel Spam eingeschränkt

Als Administrator können Sie, wenn einer Ihrer Benutzer keine E-Mails versenden darf, [die Sperre selbst aufheben](https://protection.office.com/?hash=/restrictedusers), **ohne einen Support-Fall zu eröffnen**. Der Benutzer kann innerhalb einer Stunde nach Aufhebung der Einschränkung E-Mails senden.

Der Dienst verhindert, dass ein Benutzer E-Mails sendet, nachdem er Hinweise auf ein angegriffenes Konto und/oder ausgehenden Spam entdeckt hat. Als Vorsichtsmaßnahme, führen Sie die Schritte zum [Reagieren auf ein angegriffenes E-Mail-Konto](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account) für den Benutzer aus.
