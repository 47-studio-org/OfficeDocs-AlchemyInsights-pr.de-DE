# <a name="set-passwords-to-never-expire"></a>Kennwörter läuft nie ab

Festlegen von Kennwörtern aus läuft nie ab [Sicherheit &amp; Privacy](https://portal.office.com/adminportal/home#/settings/security). Klicken Sie auf "Bearbeiten" neben Kennwortrichtlinie, und schalten Sie den Schieberegler auf "on".
  
Ausführlichere Informationen finden Sie unter: [Legen Sie die Richtlinie für den Kennwortablauf für Ihre Organisation](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
