# <a name="set-passwords-to-never-expire"></a>Festlegen, dass Kennwörter nie ablaufen

Legen Sie fest, dass Kennwörter nie vom [Sicherheits &amp; Datenschutz](https://portal.office.com/adminportal/home#/settings/security)ablaufen. Klicken Sie neben Kennwortrichtlinie auf Bearbeiten und drehen Sie den Schieberegler auf "ein".
  
Ausführlichere Informationen finden Sie unter [Festlegen der Kennwortablaufrichtlinie für Ihre Organisation](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
