# <a name="set-up-azure-active-directory"></a>Einrichten von Azure Active Directory

Das [Azure AD-Setup Handbuch](https://go.microsoft.com/fwlink/?linkid=2134390) enthält Informationen, um sicherzustellen, dass Ihre Organisation über eine starke Sicherheitsgrundlage verfügt. In diesem Leitfaden richten Sie die anfänglichen Features wie die Azure-rollenbasierte Zugriffssteuerung (Azure RBAC) für Administratoren, Azure AD Connect für Ihr lokales Verzeichnis und Azure AD Connect Health ein, damit Sie die Integrität ihrer Hybriden Identität während automatischer Synchronisierungen überwachen können.

Sie enthält außerdem wichtige Informationen zum Aktivieren von Self-Service-Kennwortzurücksetzungen, bedingtem Zugriff und integrierter Drittanbieter Anmeldung einschließlich Optionaler erweiterter Identitätsschutz-und Benutzer bereitstellen-Automatisierung.
