# <a name="setting-up-a-hybrid-environment"></a>Einrichten einer hybridumgebung

Damit der Benutzer, die nahtlos zwar zusammenarbeiten, wenn ihre Daten auf lokalen Servern oder in Office 365 gespeichert ist, müssen Sie möglicherweise eine hybridumgebung eingerichtet. Um eine Übersicht über die Optionen erhalten möchten, probieren Sie [Office 365-Integration in lokalen Umgebungen](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
