# <a name="setting-up-a-hybrid-environment"></a>Einrichten einer Hybridumgebung

Damit Ihre Benutzer nahtlos zusammenarbeiten können, unabhängig davon, ob Ihre Daten auf lokalen Servern oder in Office 365 gespeichert werden, müssen Sie möglicherweise eine Hybridumgebung einrichten. Eine Übersicht über Ihre Optionen erhalten Sie [Office 365 Integration in lokale Umgebungen](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
