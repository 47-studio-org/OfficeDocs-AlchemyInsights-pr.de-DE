# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Einrichten der Weiterleitung an mehrere e-Mail-Adressen

Navigieren Sie zuerst zum [Verwalten von Gruppen](https://portal.office.com/adminportal/home#/groups)
  
1. Erstellen einer *Verteilerliste* 
    
2. Fügen Sie mehrere Benutzer hinzu
    
3. Konfigurieren Sie anschließend die Postfach-Weiterleitung an diese Gruppe
