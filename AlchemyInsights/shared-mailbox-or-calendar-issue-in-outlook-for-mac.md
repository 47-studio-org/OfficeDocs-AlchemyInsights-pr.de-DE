# <a name="shared-mailbox-or-calendar-issue-in-outlook-for-mac"></a>FreigeGebenes Postfach oder Kalender Problem in Outlook für Mac?

Wir können Ihnen schneller helfen, wenn Sie sich direkt über die Anwendung an uns wenden. **Wechseln Sie zum Mac des betroffenen Benutzers, öffnen Sie Outlook, und klicken Sie auf \> Hilfe, um den Support zu kontaktieren**. 
  
Weitere Informationen zur Verwendung von Outlook für Mac in-App-Support finden Sie in diesem Artikel: [neue Kontakt Support Funktion in outlook 2016 für Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
