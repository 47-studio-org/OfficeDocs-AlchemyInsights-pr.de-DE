# <a name="sharepoint-files-are-missing"></a>SharePoint-Dateien fehlen

Wenn viele ihrer OneDrive-oder SharePoint-Dateien gelöscht, überschrieben, beschädigt oder von Schadsoftware infiziert werden, erhalten die Benutzer eine Vielzahl von Optionen, um den Wiederherstellungsprozess zu unterstützen.

Weitere Informationen finden Sie unter [Gewusst wie: Wiederherstellen fehlender, gelöschter oder beschädigter Elemente in SharePoint Online und OneDrive für Unternehmen](https://go.microsoft.com/fwlink/?linkid=2110774).
