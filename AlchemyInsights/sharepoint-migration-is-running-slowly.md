# <a name="sharepoint-migration-is-running-slowly"></a>SharePoint-Migration wird sehr langsam ausgeführt

Die Migrationsleistung kann durch die Netzwerkinfrastruktur, Dateigröße, Migrationszeit und Drosselung beeinträchtigt werden. Das Verständnis dieser Faktoren hilft Ihnen dabei, Ihre Migration zu planen und die Effizienz zu maximieren.

Weitere Informationen finden Sie unter:

- [Bei mir kommt es während der Migration zu einer schlechten Leistung oder Drosselung](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
