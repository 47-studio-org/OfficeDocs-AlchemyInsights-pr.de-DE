# <a name="sharepoint-new-features-announced"></a>Angekündigte SharePoint-neue Features

Bei der Ignite 2018-Konferenz kündigte Microsoft neue Innovationen für SharePoint an, darunter neue Features für moderne Portale, neue Webparts, zahlreiche Verbesserungen an Hub-Websites, News Link, Zielgruppenadressierung, Verfügbarkeit neuer mobiler apps, Verbindung mit Teams und vieles mehr...
  
Weitere Informationen zu diesen Features finden Sie im Blogbeitrag [neue Innovationen in SharePoint &amp; Microsoft 365 bieten Energie und Einfachheit für die Inhalts Zusammenarbeit.](https://go.microsoft.com/fwlink/?linkid=2026502)
  
Ihre Organisation kann an dem Zielprogramm teilnehmen, um frühzeitig Zugriff auf neue Features zu haben. Weitere Informationen finden Sie unter [Einrichten der Standard-oder Targeted Release-Optionen in Office 365](https://docs.microsoft.com/office365/admin/manage/release-options-in-office-365).
