# <a name="sharepoint-permissions-errors"></a>SharePoint-Berechtigungsfehler

Es gibt viele Szenarien, die eine dieser Nachrichten auffordern können. Am häufigsten werden Berechtigungen für den Benutzer oder Administrator falsch konfiguriert oder gar nicht konfiguriert. 

Weitere Informationen finden Sie unter "[Zugriff verweigert", "Sie benötigen die Berechtigung für den Zugriff auf diese Website" oder "Benutzer nicht im Verzeichnis gefunden" Fehler in SharePoint Online und OneDrive für Unternehmen](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business).
