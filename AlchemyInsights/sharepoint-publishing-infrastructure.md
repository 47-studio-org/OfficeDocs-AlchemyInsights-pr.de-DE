# <a name="sharepoint-publishing-infrastructure"></a>SharePoint-Veröffentlichungs Infrastruktur

> [!IMPORTANT]
> Moderne Kommunikationswebsites unterstützen nicht die klassischen Veröffentlichungsfeatures, die als SharePoint Server Veröffentlichungs Infrastruktur bezeichnet werden. Beim Versuch, das Feature für eine moderne Kommunikationswebsite zu aktivieren, wird eine Fehlermeldung angezeigt. Moderne Kommunikationswebsites bieten standardmäßig eine neuere Alternative.

Bei klassischen Websites enthält SharePoint einen klassischen Funktionsumfang, mit dem Sie Rich-Webseiten für Ihr Intranet erstellen und veröffentlichen können. Aktivieren Sie die SharePoint-Veröffentlichungs Infrastruktur auf der Websitesammlungsebene, um Sie zu aktivieren.

Weitere Informationen finden Sie unter [Aktivieren von Veröffentlichungsfeatures](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518) und [Veröffentlichungsfeatures](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)
