# <a name="sharepoint-publishing-infrastructure"></a>SharePoint-Veröffentlichungs Infrastruktur


<p>SharePoint enthält eine klassische Gruppe von Features, mit denen Sie Rich-Webseiten für Ihr Intranet erstellen und veröffentlichen können. Aktivieren Sie die SharePoint-Veröffentlichungs Infrastruktur auf der Websitesammlungsebene, um Sie zu aktivieren.</p>  <p>Weitere Informationen finden Sie unter den Links unten.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Aktivieren von Veröffentlichungsfeatures</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Veröffentlichungsfeatures</a></li>  </ul>  <p>Hinweise:</p>  <ul>  <li>Die klassischen Veröffentlichungswebsite Vorlagen enthalten standardmäßig die Veröffentlichungsfeatures.</li>  <li>Klassische Veröffentlichungsfeatures stehen auf modernen Websites nicht zur Verfügung. Moderne Websites bieten standardmäßig eine neuere Alternative.</li>  </ul>
