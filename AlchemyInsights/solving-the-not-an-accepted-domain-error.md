# <a name="got-a-parameter-cannot-be-found-error"></a>Haben Sie einen "Parameter kann nicht gefunden"-Fehler?

Wenn die Fehlermeldung **"ein Parameter kann nicht gefunden werden, der dem Parameternamen '** e-Mail-Nachrichten entspricht" angezeigt wird, bedeutet dies, dass es etwas länger dauert, bis Microsoft 365 die Einrichtung Ihres Mandanten oder Ihrer benutzerdefinierten Domäne abgeschlossen hat, wenn Sie kürzlich einen hinzugefügt haben.
  
Der Installationsvorgang kann bis zu 4 Stunden in Anspruch nehmen. Warten Sie eine Weile, damit der Setup-Prozesszeit zum fertig stellen hat, und versuchen Sie es dann erneut.
