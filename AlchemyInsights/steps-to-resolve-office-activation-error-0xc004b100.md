# <a name="steps-to-resolve-office-activation-error-0xc004b100"></a>Schritte zur Office-Aktivierung 0xc004b100 Fehlerbehebung


Führen Sie diese Lösungen, um die Office-Aktivierungsfehler 0xc004b100 zu beheben:
  
1. Führen Sie eine [Onlinereparatur von Office](https://support.office.com/article/7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).
    
2. Herunterladen Sie und installieren Sie die [Unterstützung und Recovery-Assistenten](https://aka.ms/SARA-OfficeActivation-Alchemy) und führen Sie der **Problembehandlung für die Aktivierung aus**.
