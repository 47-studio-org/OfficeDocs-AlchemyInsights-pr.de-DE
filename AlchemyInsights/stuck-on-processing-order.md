# <a name="stuck-on-processing-order"></a>Geht bei der Auftragsverarbeitung nicht weiter

Wenn Sie sich für ein Upgrade Ihres Microsoft 365-Abonnements entschieden haben, die Bestellung aber am Punkt „Ihre Bestellung wird verarbeitet“ festsitzt, lesen Sie [Dienststatus-Dashboard](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/servicehealth) Beitrag **SP207374**. Der Vorgang zum Aktualisieren Ihres Mandanten setzt die Bereitstellung voraus, die möglicherweise verzögert wird. Wir gehen davon aus, dass dieser Vorgang innerhalb von 24 Stunden durchgeführt wird. Allerdings gibt es seltene Fälle, in denen diese Anforderungen bis zu 48 Stunden dauern. Eröffnen Sie eine Serviceanfrage, wenn die Auftragsverarbeitung länger als 48 Stunden dauert.
