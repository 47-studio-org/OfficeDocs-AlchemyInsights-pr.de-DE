# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Unterstützte Version des SharePoint-Migrationstools



Das SharePoint-Migrations Tool unterstützt derzeit nur Migrationen von SharePoint 2013. Versionen 2010 und 2016 werden zu diesem Zeitpunkt nicht unterstützt.
  
Weitere Informationen finden Sie unter [Einführung in das SharePoint-Migrations Tool](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
