# <a name="sync-a-users-profile-picture"></a>Synchronisieren einer benutzerprofilbilds

Bild der Profilsynchronisierung in SharePoint Online erfordert eine Benutzeraktion, die Synchronisierung ihrer Fotos zu initiieren. Es besteht keine automatische Synchronisierung von Fotos mit SharePoint Online. SharePoint Online nur synchronisiert Fotos von Exchange Online, nicht in Azure Active Directory.
  
Wenn ein Benutzer hat eine Aktion, die ein Sync auffordern würde nie ausgeführt, oder sie wurden aktualisiert, das Profilbild und die neue eine nicht in SharePoint Online angezeigt wird, müssen der Benutzer auffordern, synchronisieren. Hier wird, wie sie dies tun können:
  
1. Öffnen Sie einen Webbrowser in einer Sitzung In privaten oder Incognito und [Wechseln Sie auf Office.com](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.office.com%2F&amp;data=02%7C01%7C%7Cbad62c504a36446096e108d614dec653%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C636719344369977864&amp;sdata=Km7ZnN8FHSouZbxOiEpQAGIKsK82SHr25uYCh3Gc%2F3s%3D&amp;reserved=0).
  
2. Melden Sie sich mit ihrem Konto arbeiten oder Schule, und klicken Sie auf die Kachel OneDrive oder SharePoint.
