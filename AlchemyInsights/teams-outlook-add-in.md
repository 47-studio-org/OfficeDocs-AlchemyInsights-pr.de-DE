# <a name="teams-outlook-add-in"></a>Microsoft Teams Outlook-Add-in

Wenn Ihre Benutzer das Microsoft Teams-Outlook-Add-in nicht sehen, nachdem Sie zu einem unterstützten Teams-Upgrademodus (TeamsOnly, SfBWithTeamsCollabAndMeetings oder Islands) wechseln, ist das Add-in möglicherweise nicht ordnungsgemäß installiert.  Lesen Sie den folgenden Artikel, um das Problem zu beheben: 

- [Warum wird das Microsoft Teams-Besprechungs-Add-in für Outlook nicht angezeigt?](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/why-can-t-i-see-the-microsoft-teams-meeting-add-in-for-outlook/ba-p/174630)
