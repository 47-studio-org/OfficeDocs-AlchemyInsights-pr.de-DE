# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Problembehandlung bei Aktualisierungsszenarien – Kachel Fehler

Warten Sie etwa 10-15 Minuten, bis die Aktualisierung in den Dashboard-Kacheln reflektiert wird. Wenn es immer noch nicht angezeigt wird, müssen Sie die Visualisierung erneut an das Dashboard anheften.
