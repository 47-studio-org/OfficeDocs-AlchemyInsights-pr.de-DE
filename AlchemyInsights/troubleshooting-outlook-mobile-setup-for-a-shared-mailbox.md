# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Problembehandlung für Outlook Mobile-Setup für ein freigegebenes Postfach

Über die auf dem mobilen Gerät installierte Outlook-App können Sie auf das freigegebene Postfach nicht zugreifen. Möchten Sie dieses Feature verwenden? Lassen Sie das Outlook-Entwicklungsteam wissen. [Stimmen Sie hier](https://go.microsoft.com/fwlink/?linked=862116)ab!
  
Sie können nur ein freigegebenes Postfach auf einem mobilen Gerät verwenden, indem Sie Ihren Browser öffnen, sich anmelden und dann zu Outlook im Internet wechseln. Anweisungen finden Sie unter [Hinzufügen eines freigegebenen Postfachs in Outlook im Internet](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Von Outlook im Web aus können Sie auf das freigegebene Postfach zugreifen.
  
Weitere Informationen: [Get in-App Help for Outlook for Ios and Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
