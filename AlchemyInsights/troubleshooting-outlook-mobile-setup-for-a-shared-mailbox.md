# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Problembehandlung für das Outlook Mobile-Setup für ein freigegebenes Postfach

Über die auf dem mobilen Gerät installierte Outlook-App können Sie auf das freigegebene Postfach nicht zugreifen. Möchten Sie dieses Feature verwenden? Informieren Sie das Outlook-Team. [Stimmen Sie hier](https://go.microsoft.com/fwlink/?linked=862116)ab!
  
Sie können ein freigegebenes Postfach nur auf einem mobilen Gerät verwenden, indem Sie Ihren Browser öffnen, sich bei Office 365 anmelden und dann Outlook im Web aufrufen. Weitere Informationen finden Sie unter [Hinzufügen eines freigegebenen Postfachs in Outlook im Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Von Outlook im Web aus können Sie auf das freigegebene Postfach zugreifen.
  
Weitere Informationen: [erhalten Sie in-App-Hilfe für Outlook für IOS und Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
