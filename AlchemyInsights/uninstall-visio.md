# <a name="uninstall-visio"></a>Deinstallieren von Visio

Downloaden und installieren Sie den [Support-und](https://aka.ms/SARA-OfficeUninstall-Alchemy)Wiederherstellungs-Assistenten.
  
1. Wenn es geöffnet wird, wählen Sie unter **Office-Produkte deinstallieren** die Version von Visio aus, die Sie deinstallieren möchten, und wählen Sie **weiter**aus. 
    
2. Fahren Sie fort, und starten Sie den Computer neu, wenn Sie dazu aufgefordert werden.
    
    Nach dem Neustart wird der Support-und Wiederherstellungs-Assistent automatisch erneut geöffnet, um den Deinstallationsvorgang abzuschließen.
    
3. Befolgten verbleibenden Ansagen, und wenn Sie ein Abonnement für Visio Online haben, können Sie Visio aus dem Office 365- [Konto Portal](https://portal.office.com/account#installs)erneut installieren. Weitere Hilfe oder Hilfe beim Installieren von Visio-fremden Versionen finden Sie unter [install Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1936). 
    
Eine Liste mit anderen Deinstallationsoptionen finden Sie unter [Deinstallieren von Office auf einem PC](https://support.office.com/article/9dd49b83-264a-477a-8fcc-2fdf5dbf61d8?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1936).
