# <a name="upgrade-to-visio-2019"></a>Upgrade auf Visio 2019

Wenn Sie die Visio-Desktop-App als Teil Ihres Visio Online-Abonnements installiert haben, haben Sie bereits die neueste Version. Informationen zum Vergleichen von Visio-Plänen und nicht-Abonnementoptionen finden Sie unter Compare Visio Options.

Wenn Sie eine nicht-Abonnementversion von Visio wie Visio 2016 haben und ein Upgrade auf Visio 2019 durchführen möchten, müssen Sie Visio 2019 kaufen oder ein Visio Online-Abonnement erwerben. Wenn Sie bereits über Visio 2019 verfügen und diese installieren möchten, lesen Sie [Installieren von Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1798). 

Wenn Sie Visio nebeneinander mit Office auf demselben Computer installieren, finden Sie weitere Informationen unter [unterstützte Szenarien für die Installation verschiedener Versionen von Office, Project und Visio auf demselben Computer](https://docs.microsoft.com/deployoffice/install-different-office-visio-and-project-versions-on-the-same-computer).

**Hinweis**: die Hauptversion von Visio 2019 liegt bei 16,0.
