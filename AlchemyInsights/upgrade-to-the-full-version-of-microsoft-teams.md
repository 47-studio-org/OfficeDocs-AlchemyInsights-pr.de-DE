# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Durchführen eines Upgrades auf die Vollversion von Microsoft-Teams

Sie können von Microsoft-Teams (kostenlos) auf die Vollversion von Microsoft-Teams, aktualisieren, wenn Sie bereit sind.
  
Die beste Möglichkeit zum Starten des Aktualisierungsvorgangs ist unten **wollen wir uns anrufen** klicken. 
  
Support-Mitarbeiter kontaktiert Sie und wählen Sie einen Office 365-Plan, geben Sie Ihre Kontakt- und Zahlung Info, wählen Sie Ihren Domänennamen Org und aktualisieren Sie die Teilnehmerliste Member einer-Hilfe.
