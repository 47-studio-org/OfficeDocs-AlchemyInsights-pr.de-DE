# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Verwenden Sie Bedingungen oder erweiterte Formeln in einem Datenfluss

Sie können einen Fluss anpassen, damit diese Aufgaben ausgeführt werden, nur, wenn eine Bedingung true ist. Info finden Sie unter [Hinzufügen einer Bedingung eines Datenflusses](https://go.microsoft.com/fwlink/?linkid=872112).
  
Um eine erweiterte Formel zu erstellen, finden Sie unter [Workflow Definition Language Schema für Azure Logik apps](https://aka.ms/logicexpressions).
