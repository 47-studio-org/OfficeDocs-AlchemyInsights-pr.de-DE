# <a name="use-the-office-offline-installer"></a>Verwenden Sie den Office-offline-installer



Müssen Sie mit dem Internet zum Herunterladen der Installer-Dateien verbunden werden, aber nach ausgeführt wurde, Sie können dann Office installieren offline auf einem PC nach Wunsch mithilfe von Office Deployment Tool (ODT)
  
ODT ist für Enterprise-Umgebungen entworfen und über die Befehlszeile wird ausgeführt, sodass die Schritte komplizierter--sind, aber sie müssen für die Installation auf einem einzigen Gerät funktionieren. Finden Sie unter [Verwendung der Office-offline-Installer](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA).
