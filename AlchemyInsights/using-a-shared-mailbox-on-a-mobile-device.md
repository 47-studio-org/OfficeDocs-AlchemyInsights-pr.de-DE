# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Verwenden eines freigegebenen Postfachs auf einem mobilen Gerät

Sie können nicht über die auf dem mobilen Gerät installierte Outlook-App auf ein freigegebenes Postfach zugreifen. Möchten Sie dieses Feature verwenden? Informieren Sie das Outlook-Team. [Stimmen Sie hier](https://go.microsoft.com/fwlink/?linked=862116)ab!
  
Sie können ein freigegebenes Postfach nur auf einem mobilen Gerät verwenden, indem Sie Ihren Browser öffnen, sich bei Office 365 anmelden und dann zu Outlook im Web wechseln. Weitere Informationen finden Sie unter [Hinzufügen eines freigegebenen Postfachs in Outlook im Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Wenn Sie über die Outlook-App auf einem mobilen Gerät auf freigegebene Postfächer zugreifen müssen, sollten Sie anstelle eines freigegebenen Postfachs eine Office 365-Gruppe erstellen. Weitere Informationen finden Sie unter [Compare Groups](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx).
  
Weitere Informationen: [erhalten Sie in-App-Hilfe für Outlook für IOS und Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
