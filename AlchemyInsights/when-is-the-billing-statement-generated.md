# <a name="when-is-the-billing-statement-generated"></a>Wann wird die Abrechnung generiert?

Jeden Monat erhalten Sie eine e-Mail, die Ihnen mitteilt, dass Ihre neue Abrechnungs Anweisung im Microsoft 365 Admin Center verfügbar ist. Ihre Rechnung enthält zwei Seiten. Seite 1 ist die Zusammenfassung der Rechnung und enthält allgemeine Informationen über die Rechnung, den Auftrag, den fälligen Betrag sowie dazu, wie Sie eine Zahlung vornehmen und wie Sie Kontakt zum Support aufnehmen können. Seite 2 enthält Details zur Abrechnungsaktivität für jedes Abonnement.
  
Weitere Informationen zu Abrechnungs Konzepten finden Sie Untergrund Legendes zu [Ihrer Rechnung](https://support.office.com/article/0724b428-fb59-4962-8c37-6674166d7507).
