# <a name="desktop-app-for-sharepoint"></a>Desktop-App für SharePoint

Ihre Organisation wird im privaten Vorschau Programm für die Desktop-App für SharePoint eingegeben. Microsoft unterstützt derzeit die Desktop-App für SharePoint (private Preview) nicht. Verwenden Sie die eingeTragenen Microsoft private Preview-Partner, um Unterstützung bei der Desktop-App für SharePoint zu erhalten.
