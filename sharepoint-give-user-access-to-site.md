# <a name="give-users-access-to-sharepoint-site"></a>Erteilen von Benutzern Zugriff auf die SharePoint-Website

<p>Mit der Zugriffs Anforderungsfunktion können Benutzer Zugriff auf Inhalte anfordern, für die Sie derzeit keine Berechtigung haben. Als Websitebesitzer können Sie das Feature so konfigurieren, dass Sie e-Mails senden, wenn jemand den Zugriff auf eine Website anfordert. Weitere Informationen finden Sie unter <a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3">Einrichten und Verwalten von Zugriffsanforderungen</a>.</p>  <p>Hier finden Sie einige häufige Links zu Zugriffsanforderungen:</p>  <ul>  <li><a href="https://support.office.com/en-us/article/-Access-Denied-to-Access-Requests-list-or-Request-approval-failed-when-you-process-a-pending-request-in-SharePoint-Online-3ec71fe1-9111-4a4c-84a5-bdfe05c2e558">"Zugriff verweigert", wenn Sie eine ausstehende Anforderung in SharePoint Online verarbeiten, um auf Anforderungsliste oder "Genehmigung fehlgeschlagen" zuzugreifen.</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow">Ändern von Einstellungen für Zugriffsanforderungen</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462557">Genehmigen oder ablehnen von Zugriffsanforderungen für eine Website</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558">Verwalten von Einladungen für externe Benutzer (nur SharePoint Online)</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260">Einladungen zurückziehen</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560">Erneutes Senden einer Einladung</a></li>  </ul>
